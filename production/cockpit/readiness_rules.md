# 共同驾驶舱准备度规则 V1

日期：2026-06-03

本规则参考 Jellyfish 的镜头准备度思想，但保持《边界回声》当前文件型工作流。

## 状态枚举

| 状态 | 含义 |
| --- | --- |
| 待拆解 | 剧本已有，但尚未拆成宫格分镜 |
| 待关键帧 | 宫格分镜已有，但关键帧未齐 |
| 待质检 | 关键帧已生成，但未完成 90 分质检 |
| 待视频 | 关键帧和 Prompt 已可用于视频生成 |
| 视频待审 | 视频已生成，等待人工审核 |
| 通过 | 当前宫格/OP 已通过审核 |
| 需重做 | 存在关键问题，需要重新生成或重写 |
| 暂停 | 暂不进入当前制作线 |

## OP 级准备度

一个 OP 可进入视频生成，必须满足：

1. 剧情锁：引用 `第一集_V4_完整剧本_最新版.md`。
2. 宫格锁：与 `第一集_OP01-OP16_主线分镜宫格表_V1.md` 的格数一致。
3. 关键帧锁：每个宫格至少有一张对应关键帧。
4. Prompt 锁：OP 段落有首帧、尾帧、动作链、负面约束。
5. 连续性锁：角色、鞋子、场景时间线、隔离门规则没有冲突。
6. 质检锁：综合自评不低于 90 分。
7. 风险备注：如需视频运动补强，必须写入 `known_risks` 或质检记录。

## 宫格级准备度

一个宫格可进入视频生成，必须满足：

- `keyframe_path` 存在。
- `assets` 全部能在 `asset_registry.json` 中找到。
- `status` 为 `待视频` 或更高。
- `qa_score` 不低于 90；如果继承 OP 综合评分，必须标注 `qa_source`。
- `continuity_checks` 全部为 `true`。

## 当前 OP10-OP12 特别规则

### OP10

- 必须看出妈妈松手是保护性松手。
- 隔离门必须两米以上并真正隔开两侧。
- 小夏右脚必须是白袜，左脚单鞋。

### OP11

- 妈妈情绪可以激动，但不能慌张失控。
- 必须表现敲门、找应急开关、隔门安慰。
- OP11-4 关键帧不是应急开关硬特写，因此视频运动必须补强。

### OP12

- 妈妈被人潮带走时仍朝向小夏。
- 坍塌/电气爆闪吞没后半句，不做大火球，不血腥。
- 小夏必须先停顿，再哭着跑向隔离门。
- OP12-8 必须无人，并为 OP13 同机位腐蚀做桥。
