# ComfyUI Video Demo Visual Identity

## Style Prompt
Cold cinematic tutorial for the AI live-action drama "Boundary Echo". The image should feel like a production control room at night: dark graphite canvas, restrained cyan system light, pale amber human memory light, and red alert accents used only for warnings. The tone is practical and beginner-friendly, but still belongs to a mystery sci-fi drama workflow.

## Colors
- Background: `#091014`
- Panel: `#111C20`
- Soft panel: `#18252A`
- Text: `#E8F1EE`
- Muted text: `#9DB0AA`
- Cyan accent: `#72D7D0`
- Amber memory: `#D9A85A`
- Alert red: `#D05D57`

## Typography
- Chinese and main UI text: Microsoft YaHei UI / Microsoft YaHei
- File paths and node labels: Consolas
- Use heavy title weights and lighter body weights for clear video hierarchy.

## Motion
- Slow focus-pull and blur crossfade feel between scenes.
- Use subtle push-in on production stills.
- Use cyan path highlights for workflow logic and amber highlights for story continuity.

## What NOT to Do
- Do not use bright game-like neon everywhere.
- Do not cover keyframe subjects with large captions.
- Do not use jump cuts between tutorial chapters.
- Do not make the video feel like a generic software ad.
