01|这一版专门讲 ComfyUI 界面上你会用到的按钮和功能。目标很简单：新手看完以后，知道点哪里、改哪里、怎么从一张关键帧开始做《边界回声》的视频短段。
02|先从启动开始。以后如果 ComfyUI 没打开，就双击 F 盘 ComfyUI 文件夹里的“启动 ComfyUI”。浏览器里能看到 127.0.0.1 冒号 8188，就说明界面已经连上本机服务。
03|整个界面先分成四块看：左边是功能栏，中间是画布，画布上是节点，右边通常是队列、历史、设置或节点属性。新手不要急着找全部按钮，先记住这四块。
04|左侧功能栏里，最常用的是节点、模型、工作流、应用、模板。节点用来找功能块，模型用来管理模型，工作流用来保存和加载流程，模板可以快速打开官方示例。
05|中间画布就像一张线路图。鼠标滚轮缩放，按住鼠标中键或者空格拖动画布。点住节点标题可以移动节点，框选后按删除可以删掉不需要的节点。
06|每个节点左右两边的小圆点叫端口。左边是输入，右边是输出。你要做的事，就是把上一节点的输出，拖线接到下一节点的输入。线接错了，生成就会报错。
07|出图最基础的节点是 Checkpoint Loader。它负责选择模型。这里要点模型下拉框，选择 v1-5-pruned-emaonly-fp16.safetensors。以后换别的模型，也是在这里换。
08|正向提示词节点，一般叫 CLIP Text Encode。点击文本框，把你想要的画面写进去。做《边界回声》时，要写真人剧、电影级关键帧、同机位、废站、儿童鞋、克制情绪。
09|负向提示词节点也很关键。这里写不想要的东西，比如 anime、cartoon、changed camera angle、station lights、neon ads、large white hair、watermark、bad face。
10|KSampler 是生成核心。新手先只改几个参数：Steps 设二十左右，CFG 设七到八，Sampler 先用 euler，Scheduler 先用 normal，Denoise 出图用一，图生图或视频会再调整。
11|Empty Latent Image 决定画面大小。新手先用 512 乘 512 或 768 乘 432。Batch size 先用一。不要一开始就开很大，不然显存压力和等待时间都会变大。
12|点击 Queue Prompt 就会开始生成。按钮通常在右侧或底部队列区域。点下去以后，不要连续乱点，先看进度条和队列，等当前任务跑完。
13|生成完成后，看输出节点或右侧历史。图片会保存到 F 盘 ComfyUI 的 output 文件夹。每次测试都建议改文件名前缀，比如 OP13_test，这样后面好复盘。
14|要做视频，基础 ComfyUI 还不够。先点 Manager，也就是 ComfyUI-Manager。进入 Custom Nodes Manager，搜索 Video Helper Suite 和 AnimateDiff。安装后重启 ComfyUI。
15|装好视频节点后，图生视频的基本结构是：Load Image 导入关键帧，正向提示词，负向提示词，视频模型或运动节点，最后接 VHS Save Video 之类的保存视频节点。
16|做《边界回声》时，先用 OP13-1 这张同机位废站鞋做短测。Load Image 选择这张图，提示词只写轻微环境运动、湿冷废站、鞋子不动，不要写大动作。
17|视频参数先保守：时长三到五秒，帧率十二到二十四，运动幅度低，分辨率先低。第一轮只检查同机位、背景、角色和道具有没有跑掉。
18|最后记住一个新手顺序：先保存工作流，再 Queue Prompt，再看历史输出，再抽帧复盘。确认小段稳定以后，才扩展到 OP13-1 到 OP13-3，最后再做整段 OP13 到 OP16。
