# 《边界回声》世界观与地图说明

## 用途

本文件用于建立《边界回声》的空间逻辑，帮助后续决定每个单元事件发生在哪里、为什么角色会走到那里、以及主线如何逐步靠近静海系统核心。

## 世界观一句话

静海系统原本是城市级灾后修复与记忆治疗系统。静海断层后，它把人的遗憾、失踪记录、未完成愿望和现实空间压在一起，形成异常区。主角团进入异常区，不是为了消灭怪物，而是找到遗憾源点，让被困住的记忆能够继续向前。

## 空间分层

### 现实城区

城市仍在运转的部分。这里有调查、档案、幸存者、现实证据和官方记录。陆衡、林见舟等角色的现实身份从这里进入故事。

### 异常区边缘

现实和回声重叠的缓冲带。街道会漂移，门牌会失真，雾墙会吞掉正常路径。澪星的引航能力在这里最有用。

### 记忆回声区

被遗憾循环占据的具体地点。每个单元事件都有一个“遗憾源点”，它通常不是最显眼的物件，而是某个人最放不下的一句话、一次等待、一次选择。

### 静海系统深层

静海断层更接近核心的地方。这里不是普通废墟，而是静海系统残留协议、损坏数据库、样本编号和人形锚点真相交叠的区域。B-07、陆星遥碎片、林见舟校准记录会逐步指向这里。

## 地图坐标使用法

地图上方用 A-H 标出横向区域，左侧用 1-6 标出纵深层级。横向越靠右，越接近静海系统深层；纵向越靠下，越偏向废城生活、轨道和地下设施。

后续讨论时可以直接说：

- “第三集的林见舟线放在 B2 到 C3，靠现实证据区。”
- “许明烬第一次正式登场放在 C5 到 D5，靠黑市维修点。”
- “主线危险事件放在 F3 到 G4，靠地下轨道和静海核心。”

## 后续放置事件的规则

1. 每个事件先确定“现实地点”，再确定“遗憾源点”。
2. 遗憾源点不要总放在最显眼的物件上。鞋、车、电话、屏幕只是入口，真正源点通常是某个人误解了爱、等待、选择或失踪。
3. 越靠近异常区边缘，事件越适合做单元情感故事。
4. 越靠近静海系统深层，事件越应该牵动 B-07、陆星遥、林见舟校准记录、许明烬最后一站等主线。
5. 澜冰屿每完成一次回收，地图上应出现一条“坐标布条褪白”的代价记录。

## 当前地图版本判断

这是 V1 空间草案，不是最终城市规划图。它的目的不是地理完全精确，而是让创作团队先拥有稳定的方向感：

- 旧北站是观众入口。
- 废弃派出所是陆衡入口。
- 北湾总站可作为第5集团队分工入口。
- 静海系统核心暂时保持远处不可达。
