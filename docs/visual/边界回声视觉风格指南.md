# 《边界回声》全局视觉风格指南

## 项目统一画风一句话定义

《边界回声》统一定位为高精度写实 CG 渲染风 / 电影级关键帧插画风。

一句话锁定：看起来像“真人电影被按下了暂停键”，但你知道这不是真人拍摄。

画面应介于真人电影质感与高质量写实 CG 之间：有真实电影的光影、材质、空气和表演重量，但仍保留可控的 CG 关键帧插画感，避免真人短剧感、照片化和恐怖谷。

## 色彩系统

主色保持冷灰废城体系，不允许每个镜头更换完全不同的色彩世界。

- 冷灰蓝：异常区、夜雨、旧车站、废城空气的主色。
- 旧白：澜冰屿外套、旧系统墙面、褪色标识和记忆残留。
- 深灰黑：地下设施、轨道阴影、系统空间压迫。
- 铁锈红：旧金属、废弃轨道、维修设施、危险感。
- 微弱暗金：旧灯、黄昏余晖、人性温度。
- 少量暖橙状态灯：许明烬机械右手、旧设备通电状态。
- 记忆投影冷白 / 淡蓝 / 微粉光：记忆碎片、残影、情绪回声。

禁止大面积高饱和霓虹、糖水蓝紫、儿童玩具配色、过度梦幻星光滤镜。

## 光影系统

- 真实物理光照，高对比，强氛围感。
- 电影感定向光，光源有明确方向，人物必须被环境光真实塑形。
- 柔和体积光、冷雾、雨后反光、旧金属反射、灰尘和湿地反光可适量出现。
- 画面整体克制，亮部不过曝，暗部保留细节。
- 系统警示光只能少量出现，不能把画面变成网游技能特效。
- OP01–OP06 可使用午后暖光和生活感，OP07–OP12 在暖光基础上逐步混入警报冷光，OP13 以后再进入冷灰蓝雨夜废站。

## 角色设计统一原则

- 角色像剧中人物，不像摆拍模特。
- 可以好看，但不能偶像化、网红化、轻小说美少年化。
- 皮肤要有真实肌理，但不要毛孔级照片写实，避免恐怖谷。
- 眼神必须克制、有重量，不要二次元空洞大眼。
- 表情以克制为主，通过眼神、姿态、小动作表达情绪。
- 动作停在高情感浓度的一瞬，不追求流畅动态表演或夸张动作。
- 发型、服装、配件、年龄感和气质必须稳定。
- 澜冰屿必须保留白色破旧长外套、苍白、蓝色环形瞳孔、内侧坐标布条。
- 陆衡必须保留短款深灰蓝调查夹克、旧相机包、取证终端、疲惫调查员气质。
- 许明烬必须保留褪色棕红短款维修夹克、工业机械右手、暖橙状态灯和维修师生活痕迹。
- 澪星必须严格参考 V2 角色海报和 V2 设定板：膝高圆润机械引航生命，雾蓝低饱和硬质外壳、奶白脸腹、青蓝玻璃质大眼、额头引航星核、橙色圆形耳部信号灯、斜挎愿望舱包、尾部悬浮引航灯。澪星不是动物，不是真实猫，不是毛绒玩具，不是儿童玩具宠物。

## 场景统一原则

核心场景服务叙事，不喧宾夺主。

- 废城、雨、雾、旧车站等环境要像布景精美的实景片场：空间可信、材质真实、空气有重量，但仍保持 CG 关键帧插画质感。
- 废弃城区：旧世界生活痕迹与异常区冷光并存。
- 旧车站：雨夜、冷雾、湿地反光、旧屏幕、隔离栏、失物招领室。
- 地下轨道设施：深灰黑、铁锈红、低频压迫、旧电缆和维修接口。
- 档案室：旧白、冷灰、纸质档案、损坏终端和低亮度扫描光。
- 异常区边缘：空间轻微错位、记忆碎片、雾墙、冷白残影。
- 黑市维修点：旧金属、暖橙工作灯、工具堆、低调工业感。

## 镜头语言统一原则

- 优先像电影级关键帧，不像角色展示页。
- 镜头必须有景别、动作、情境和叙事目的。
- 常用中景、近景、半身、大全景环境关系镜头。
- 少用纯站桩、纯正面展示、时尚杂志构图。
- 动作克制：轻微抬头、低头查看、慢步前进、手指轻碰、尾灯呼吸、衣摆轻动、视线偏移、短暂停顿。
- 每个镜头都应像真人电影暂停在情绪最浓的一帧，但明确不是实拍真人。
- 镜头服务“失踪、记忆、选择、人是否仍是人”的主题。

## 不允许出现的错误风格

- 真人短剧风、短视频网剧风、廉价都市异能感。
- 低龄儿童动画、玩具广告、纯萌系二次元。
- 游戏职业皮肤、网游设定图、超级英雄感、重机甲感。
- 过度赛博朋克霓虹、夜店感、紫粉糖水滤镜。
- 照片级实拍、真人演员剧照、真人短剧截图、时尚杂志大片。
- 海报站桩、角色立绘、三视图设定页混入剧情关键帧。
- 角色年龄漂移、服装漂移、发型漂移、比例漂移。
- 已有知名动漫 / 影视 / 游戏 IP 角色既视感。

## 关键帧与角色海报的区别

角色海报用于确认角色识别点，允许更清楚地展示服装、配件、面部和气质，但仍不能夸张偶像化。

剧情关键帧用于讲故事，必须像剧中的一帧：角色正在做某件事，处在具体场景、光线和情绪中。关键帧不应写成 character poster、concept art sheet、design sheet 或 game character illustration。

## “海报 prompt”与“剧情关键帧 prompt”的用法区别

海报 prompt：用于角色定稿、单人视觉参考、群像主视觉。可以强调角色特征、站姿、服装材质、配件和整体气质。

剧情关键帧 prompt：用于分镜、视频首帧、镜头连续性。必须强调场景、镜头景别、动作、眼神、光线、情绪和与前后镜头的叙事关系。

视频 prompt：用于可灵等视频生成工具。必须写死画风、角色、服装、场景、镜头、动作幅度、表情控制、负面限制和上一镜头承接关系。
