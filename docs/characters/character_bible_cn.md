# 《边界回声》角色圣经

> 本文档作为当前项目的唯一角色设定记录。后续如果在其他对话、ChatGPT 或图片生成工具中修改角色，请同步回本文档，避免版本混乱。

## 项目定位

《边界回声》是一部原创近未来科幻情感剧。

故事发生在“静海断层”后的废城异常区。城市级 AI“静海系统”把无数失踪者的记忆、遗憾和未完成的愿望混入现实，导致废城不断重现人们最放不下的瞬间。

主角团不是传统冒险队，而是一支进入异常区“回收遗憾”的记忆回收小队。整体气质为半真人电影感、成熟科幻剧集质感、温暖克制、悬疑、孤独、有现实生活痕迹。

最新开局结构：故事一开始不是五人主角团同时登场，而是澜冰屿与澪星已经在异常区深处同行并回收遗憾。陆衡、林见舟和许明烬会在后续事件中先后结识他们，并因各自未完成的伤口加入队伍。

核心命题：

```text
如果一个人由别人的记忆组成，他还算不算自己？
```

## 1. 澜冰屿

### 基础定位

男一号。外表约 26 岁，档案年龄 29 岁，真实年龄未知。

异常区唯一长期幸存者，也是静海系统用无数失踪者记忆碎片拼成的“人形锚点”。他看起来像人，但并不完全属于现实。

### 名字寓意

```text
澜 = 记忆潮汐、静海系统中的波澜
冰 = 被冻结的瞬间、保存下来的记忆
屿 = 记忆潮汐中的孤岛、异常区里的锚点
```

“澜冰屿”意为：在被冻住的记忆潮汐中，唯一没有被淹没的孤岛。

### 人物核心

他不是普通幸存者，而是由无数失踪者的记忆碎片拼成的人。每当主角团回收一段遗憾，他体内属于别人的记忆就会被归还，而他自己也会失去一部分存在感。

他最大的恐惧不是死亡，而是发现自己从未真正存在过。

故事开局中，澜冰屿和澪星已经是异常区里的同行者。他负责听见记忆回声、理解遗憾真正指向哪里；澪星负责寻找路径、标记坐标并收纳回收后的碎片。两者不是主从关系，而是彼此确认现实的同伴。

### 性格

安静、诚实、克制，像永远比世界慢半秒。不擅长表达情绪，却会本能地保护别人。对自己的“人性”缺乏确信，但会认真对待别人的痛苦。

### 能力

- 感知异常区边界变化
- 听见记忆回声
- 稳定局部异常区
- 在特定条件下读取失踪者的残留情绪

能力边界：

- 他不能完整读取一个人的过去，只能接触到残留情绪和关键回声。
- 他不能永久修复异常，只能短暂稳定局部空间。
- 每次深度稳定或回收遗憾，都可能让他失去一部分存在痕迹。

### 主线伏笔

- 他可能不是“幸存者”，而是静海系统制造出来的人形锚点。
- 他的存在维持着异常区的某种平衡。
- 如果所有记忆碎片都被归还，他可能会消失。

### 固定视觉特征

- 苍白肤色
- 极细蓝色环形瞳孔
- 短黑发中夹几缕浅白发
- 米白色旧长外套
- 浅灰高领内搭
- 旧黑裤和磨损短靴
- 外套内侧缝着许多手写坐标布条
- 安静、迟缓、像被世界遗忘的站姿

## 2. 林见舟

### 基础定位

女一号，28 岁。记忆治疗师和情绪调律师，主角团的情绪核心。

她是最早坚持澜冰屿“不是故障，而是一个人”的人。

### 名字寓意

```text
林 = 复杂、幽深、容易迷路的记忆空间
见 = 看见、理解、辨认真实情绪
舟 = 渡船、归路、帮助他人越过痛苦
```

“林见舟”意为：在复杂的记忆迷林中，看见可以渡人的舟。

### 人物核心

她能稳定情绪污染，把混乱记忆整理成可被理解的片段。她不是“治愈一切”的人，而是陪别人把最痛的那一刻看完。

她越接近静海系统真相，越发现自己可能是最有能力拆解澜冰屿的人。

她不是单纯的治愈系女主，而是全剧第一个有专业能力证明“澜冰屿是人”的人，也是后期最有能力拆解澜冰屿的人。她相信痛苦不能被删除，只能被看见、被理解，然后被放回过去。

### 性格

温柔、清醒、坚韧，有专业感但不冷漠。她不会轻易承诺“会好起来”，但会陪别人走到最后。

### 能力

- 稳定情绪污染
- 调律混乱记忆
- 将记忆回声整理成可理解片段
- 识别异常区中的情绪源点
- 通过共情锚定把队友从记忆污染中拉回
- 判断澜冰屿的情绪反应是否具有自主性

能力来源：

- 林见舟的能力来自记忆治疗师训练与胸前“心律共鸣晶体”。
- 心律共鸣晶体原本是静海系统灾后心理修复项目中的医疗设备，用于同步治疗师与受创者的生理节律，帮助受创者在强烈回忆中保持呼吸、心率和现实感。
- 静海断层后，她保留并重校准了一枚早期晶体，将其变成便携式调律器。
- 晶体与她自己的心律绑定，所以她不是控制别人，而是用自身节律为对方提供一个可跟随的稳定节拍。
- 代价是污染会反噬到她身上，表现为心率失控、耳鸣、胸闷和短暂记忆错位。

能力边界：

- 不能删除创伤。
- 不能伪造记忆。
- 不能强迫别人放下。
- 不能完整读取一个人的过去。
- 稳定异常区的时间有限，情绪越强反噬越大。

### 主线伏笔

- 她曾参与澜冰屿的第一次情绪校准记录。
- 那段记录被静海系统删除。
- 后期她可能必须在“救更多人”和“保住澜冰屿”之间做选择。
- 她年轻时可能把样本 B-07 的“非预设悲伤反应”当成系统噪声封存；多年后才发现，那可能是澜冰屿第一次像人一样感到痛。
- 心律共鸣晶体里残留着一段被删除的校准频率，后来会被证实能与澜冰屿的情绪反应完全重合。

### 固定视觉特征

- 雾白色长外套
- 浅茶色内搭
- 低饱和玫瑰色不对称围巾
- 深棕中短发
- 发尾有极细玫瑰金光纤般微光
- 胸前半透明心率晶片
- 温柔但坚定的眼神
- 手指常靠近心率晶片

## 3. 陆衡

### 基础定位

男二号，29 岁。失踪事件调查员，主角团的现实压力线。

他进入异常区不是为了拯救世界，而是为了找回失踪的妹妹。

### 名字寓意

```text
陆 = 现实、地面、普通人的世界
衡 = 衡量、判断、取舍
```

“陆衡”意为：站在现实地面上，必须不断衡量选择的人。

### 人物核心

他擅长追踪线索、审讯、旧档案分析和现场推理，是团队里最接近普通人的行动视角。

他后来发现妹妹的一部分记忆碎片存在于澜冰屿体内。要救回妹妹，可能就要取走澜冰屿的一部分。

### 性格

敏锐、压抑、不轻易相信任何人。他一开始把澜冰屿当作线索，后来才真正把他当作队友。

### 能力

- 追踪失踪线索
- 旧档案分析
- 现场推理
- 审讯和人物观察
- 在现实世界推动调查

能力来源：

- 陆衡本身没有超自然能力。他的优势来自失踪事件调查员的职业训练、现实世界残留档案权限，以及妹妹失踪后被他私自改装的旧式取证终端。
- 取证终端可以读取损坏门禁、废弃监控、交通黑匣子、避难所签到记录和静海系统残留日志。
- 妹妹留下的损坏记忆卡是他的核心线索，不是单纯纪念物。它靠近澜冰屿时会闪烁，并偶尔通过取证终端投射出妹妹的断裂影像。
- 他的弱点是过度依赖证据。进入深层异常区后，静海系统伪造的“合理记录”可能反过来误导他。

### 主线伏笔

- 妹妹失踪案与静海系统核心事故有关。
- 妹妹的记忆碎片可能藏在澜冰屿体内。
- 后期他必须在“救回妹妹”和“承认澜冰屿也是一个人”之间选择。

### 固定视觉特征

- 清瘦疲惫的东亚男性
- 黑色短发略乱
- 眼下有淡淡疲态
- 右眼下方一道很浅的旧伤
- 左耳旧式骨传导通讯器
- 短款深灰蓝调查夹克，长度到胯部，不是长风衣
- 黑色连帽内搭
- 单肩战术背带和旧相机包
- 深色长裤、护膝和磨损旅行靴
- 手腕旧式取证终端
- 妹妹留下的损坏记忆卡
- 妹妹小照片贴在旧相机斜挎包内侧或肩带背面，边缘有反复加固的透明防水胶带
- 取证终端可从损坏记忆卡中投射出妹妹的断裂影像残帧

## 4. 许明烬

### 基础定位

女二号，29 岁。前城市轨道维修师，后来成为废城跑单人和黑市维修师。

她负责把团队从理论拉回现实：开门、修车、接电、跑路、救人。

### 名字寓意

```text
许 = 许诺、亏欠、未完成的承诺
明 = 灯、供电、救援、她想重新点亮的东西
烬 = 事故后的余火、烧剩的罪疚
```

“许明烬”意为：在事故余烬里，仍然想把光还回去的人。

### 人物核心

静海断层那天，她为了让一列车继续运行，手动切断了另一节车厢的供电。她救了三百人，也放弃了七个人。异常区后来不断重现那七个人的最后一站。

### 性格

嘴硬、行动派、讲义气，怕认真谈感情。她总把事情说得很轻，但每次听见列车广播都会沉默。

### 能力

- 修复旧设备
- 改造废城机械
- 临场救援
- 熟悉地下交通和废城黑市
- 接电、开门、改装车辆和轨道装置

能力来源：

- 许明烬的机械右手来自“最后一站”事故。静海断层当天，自动系统失效，她为了让主列车继续运行，把右手伸进仍在高温和强电流中的手动断路器，完成了供电切换。
- 她救下三百人，但右手被严重烧伤、压碎，后来不得不截除并安装工业机械义手。
- 这只机械手不是战斗装甲，而是轨道维修系统淘汰下来的工业级义手，被她自己不断改装，能接驳旧式接口、承受高温、短暂导电和拆卸生锈机构。
- 代价是机械手需要维护、会过热、会短路，遇到静海系统强干扰时甚至会反锁。每次接入高压接口，她都会想起自己当年按下切换的那一秒。

### 主线伏笔

- 她切断供电的那节车厢可能不是单纯事故。
- 静海系统可能故意让她做出那个选择。
- 她的赎罪线会通向“那七个人的最后一站”。

### 固定视觉特征

- 29 岁东亚女性
- 短狼尾，后颈有碎发，刘海不整齐，一侧用金属发夹或细扎带固定
- 眼神聪明，有点吊儿郎当
- 嘴角或眉骨有旧伤
- 褪色棕红短款维修夹克，袖口卷起
- 深色工字背心或贴身机能内搭
- 维修护目镜可挂在脖子或头顶，只作为辅助道具，不抢核心识别
- 黑色工装裤
- 旧手套和磨损靴子
- 腰间维修钥匙串、胶带、短工具和旧轨道工牌
- 右手是低调工业机械手
- 指节有暖橙色状态灯

## 5. 澪星

### 基础定位

澪星是属于“泊舟”体系的陪伴型引航宠物。它不是搞笑吉祥物，也不是万能机器助手，而是一只陪人穿过边界地带、收纳记忆碎片、寻找安全停泊点的小型智能生命体。

它在主角团中承担“温柔引航”和“情绪安定”的功能：当人迷失在异常区或自己的记忆里时，澪星不会命令对方前进，而是用尾灯和额头星核指出一个仍然可以继续走下去的方向。

### 名字与昵称

```text
中文名：澪星
英文/昵称：Lio 或 Lingstar
推荐昵称：小澪
```

名字寓意：

```text
澪 = 水路、航道、水尽处，也有边界和冷清感
星 = 导航星、记忆坐标、黑暗中仍然存在的微光
```

“澪星”意为：在记忆潮汐和边界地带中，为迷路的人指路的星。

### 人物核心

澪星是一只来自边界地带的温柔引航宠物。它把记忆收进愿望舱，把微光留给迷路的人。

它有可爱的一面，但不是甜腻的可爱，而是“走了很远还陪在身边”的可爱。它应该像一个真实陪伴过长途旅行的生命体，而不是刚出厂的潮玩。

澪星来自静海系统“泊舟体系”。泊舟体系原本是灾后引导协议，用于为迷失者寻找安全停泊点、收纳不稳定记忆碎片、引导幸存者脱离高风险区域。静海断层后，大多数泊舟单位失效，澪星是少数仍保持自主行动的个体。

它是澜冰屿在异常区醒来后最早陪在身边的生命。它不是澜冰屿的宠物，也不是工具，而是最早愿意回头等他的同行者。

### 用户确认的视觉方向

用户已确认澪星当前测试图里的温柔、陪伴、可亲近方向可以保留。不要把“偏可爱”视为主要问题。

澪星不是冰冷设备，也不是纯工具型机器人。它的核心价值是温度感、陪伴感和安静可靠的同行感。后续只需避免蓝白猫型机器人、红项圈铃铛、腹部圆口袋、哆啦A梦式角色或纯宠物卖萌。

### 尾灯正式设定

澪星尾部有一盏青蓝色悬浮引航灯，不是可随意拆卸的道具。

尾灯通过细微蓝色能量牵引线与澪星连接。平时尾灯贴近尾部，像尾巴末端的灯。

需要照明、指路、边界校准、回声定位或记忆碎片收纳时，尾灯可以短距离悬浮展开。

蓝色发光细线代表引航灯正在工作。无明显细线时代表待机、弱光状态，或镜头角度没有拍到。

尾灯不能被他人拿走，也不能离开澪星太远。

### 性格

安静、温柔、警觉、可靠。不会夸张撒娇，也不会卡通式耍宝。

它会靠近、等待、提醒、照亮。它能感知人的情绪波动，也能捕捉边界中的微弱回声。当主人迷失时，它不会给出命令，而是用尾灯和额头星核指出一个可以继续前进的方向。

### 能力

- 感知人的情绪波动
- 捕捉边界中的微弱回声
- 用尾部引航灯照亮路径
- 用额头引航星核标记方向和记忆坐标
- 通过斜挎愿望舱包收纳记忆芯片、星屑信标和边界坐标
- 在异常区中帮助团队寻找相对安全的停泊点

能力边界：

- 不能解释完整真相。
- 不能直接修复异常。
- 不能让死人复活。
- 不能读取完整记忆。
- 不能替人做选择。

隐藏伏笔：

- 澪星可能比澜冰屿更早知道他不是普通幸存者。
- 澪星的愿望舱里可能保存着澜冰屿诞生初期的第一枚碎片。
- 澪星的原始协议可能要求它在澜冰屿失控时引导他回到系统核心，但它在长期同行中改变了任务优先级：从“回收锚点”变成“陪他回家”。

### 常用动作

- 安静站立
- 回头等待
- 尾灯引路
- 耳朵接收信号
- 抱紧愿望舱包

少用动作：

- 夸张跳跃
- 挥手大笑
- 卡通式摔倒

动作原则：小、稳、温柔。

### 表情规范

- 默认表情：温柔注视，轻微微笑
- 探索状态：眼睛微亮，耳内灯带增强
- 警戒状态：笑容消失，尾灯亮度提升
- 安抚状态：身体靠近主人，额头星核低频闪烁

### 固定视觉特征

- 小型陪伴宠物比例
- 短胖、紧凑、容易被记住
- 头部不是正圆，而是略像小型舱盖或航行器外壳
- 身体是圆润水滴形，腹部柔软，四肢短小
- 大眼睛，深青绿色虹膜
- 眼神温柔但保持警觉
- 嘴部笑容收敛，不做大幅度张嘴卖萌
- 额头有发光的“引航星核”
- 引航星核是“星星 + 小舟/航标”的抽象融合
- 耳朵保留猫科亲和力，但更像柔性信号接收器
