# 第一集《尾灯照见的鞋》V3 关键帧与视频化计划

版本：情感强化版 V3  
状态：待审阅  
用途：基于第一集《尾灯照见的鞋》情感强化版 V3，将旧版按事件推进的关键帧计划，升级为按情感推进的关键帧和视频计划。

---

## 1. V3 制作原则

第一集 V3 不再以旧版 S01-S38 的事件顺序作为直接制作主轴，而是以观众情绪推进为主轴。每个关键帧先回答“观众在这一刻应该感受到什么”，再决定画面、动作和视频化方式。

核心原则：

- 情感镜头优先于设定镜头。
- 人物动作优先于空间奇观。
- 小夏的等待优先于系统异常。
- 澪星担心澜冰屿优先于单纯发光带路。
- 澜冰屿失去回家路优先于单元任务完成。
- 旧版 S18-S24 样片流程暂缓，不再直接作为当前制作样片。
- 不再把每个关键帧都当成图生视频任务。
- 系统文字、B-07 警告、广播切断、屏幕 UI 统一交给后期，避免交给 AI 直接生成乱码。

第一集 V3 的制作目标不是展示废站设定，而是让观众记住：

```text
小夏不是不想离开。
她只是害怕妈妈回来找不到她。

澜冰屿不是单纯完成任务。
他是在把别人没听见的后半句话还回去，同时失去自己的一部分回家路。

澪星不是工具。
它是第一个发现澜冰屿正在变少的陪伴者。
```

## 2. K01-K15 核心关键帧清单

### K01 妈妈给小夏系鞋带

剧情功能：建立儿童鞋不是普通遗物，而是母亲承诺的残留。  
观众情绪目标：让观众先感到这是一段普通、具体、温柔的爱。  
画面动作：小夏坐在旧车站长椅边，低头看散开的鞋带。妈妈蹲下替她系紧鞋带，拉平鞋带结，轻轻拍一下鞋面，再摸摸小夏的头。  
视频化建议：静态动效为主，可补轻微手部动作。  
成本等级：低到中。

### K02 澪星尾灯照见儿童鞋

剧情功能：从温暖记忆切回冷雾现实，引出本集遗憾入口。  
观众情绪目标：让观众感到这只鞋已经被留在这里很久。  
画面动作：澪星停在湿漉漉的儿童鞋旁，尾灯压低，照见散开的鞋带和积水里的微弱倒影。澜冰屿从雾里走近，看到鞋后停步。  
视频化建议：静态动效，用尾灯扫光、雾层、积水反光完成。  
成本等级：低。

### K03 水面倒影里的小夏反复系鞋带

剧情功能：说明鞋不是遗物，而是进入等待循环的入口。  
观众情绪目标：让观众第一次看见小夏被困在一个重复动作里。  
画面动作：澜冰屿半蹲，手悬在水面上方。倒影里的小夏反复蹲下系鞋带，每次快站起时动作卡住。澪星愿望舱收纳失败。  
视频化建议：静态动效，用水面波纹和重复闪回完成。  
成本等级：低。

### K04 小夏的等待规则

剧情功能：把“等待”具体化成可见、可重复、让人心疼的动作规则。  
观众情绪目标：让观众明白小夏不是傻等，她是在用孩子能理解的方式守住妈妈的话。  
画面动作：小夏抱着小布包，想迈步又退回黄线后；她反复坐回同一张长椅，听见广播就低头看鞋。澜冰屿远远观察，没有打断。  
视频化建议：静态动效或蒙太奇快切。  
成本等级：低。

### K05 事故当天隔离门落下

剧情功能：交代误解的来源，小夏只听见“站在那里等我”，没听见“我一定会绕回来”。  
观众情绪目标：让观众知道小夏不是被抛弃，而是后半句承诺被灾难和广播切断。  
画面动作：隔离门落下，小夏收回迈出去的脚。妈妈在门另一侧伸手，嘴型仍在说话，但声音被广播吞掉。澜冰屿站在记忆之外，下意识伸手却碰不到过去。  
视频化建议：静态动效加声音设计，隔离门瞬间可后期完成。  
成本等级：中。

### K06 澜冰屿半蹲面对小夏

剧情功能：澜冰屿第一次尝试直接带小夏离开，却被小夏的恐惧击中。  
观众情绪目标：让观众明白小夏不是不能走，而是不敢走。她怕妈妈回来找不到她。  
画面动作：澜冰屿慢慢走近小夏，半蹲到她的视线高度，伸出手。小夏抬头看他，眼里有一点希望，却又低头看鞋，说出担心。澜冰屿伸出的手停在半空。  
关键台词：  
澜冰屿：“你不用一直等在这里。”  
小夏：“可是……妈妈回来找不到我怎么办？”  
视频化建议：图生视频。  
成本等级：高。

### K07 系统识别澜冰屿

剧情功能：系统把澜冰屿识别为未登记人员，开始把他拖入等待循环。  
观众情绪目标：让观众意识到澜冰屿不是旁观者，他也会被这个遗憾吞进去。  
画面动作：冷光扫过澜冰屿，闸机、隔离栏和屏幕转向他。他低头看见自己的影子被拉长、变浅，手指短暂绷紧。  
视频化建议：静态动效加后期文字、屏幕闪烁、低频音效。  
成本等级：低到中。

### K08 澪星尾灯稳住澜冰屿

剧情功能：澪星第一次真正表现为情感保护者，而不是工具。  
观众情绪目标：让观众感到澪星在救澜冰屿，也在担心他被遗忘。  
画面动作：澪星迅速靠近澜冰屿脚边，把尾巴从身后绕到身前，尾灯贴近地面形成柔和青蓝光圈，照住澜冰屿变浅的影子。澜冰屿绷紧的手指慢慢放松，低声确认自己还在。  
关键台词：  
澜冰屿：“我还在。”  
视频化建议：图生视频。  
成本等级：高。

### K09 澜冰屿发现源点不是鞋

剧情功能：从“找鞋”转向“找回被切断的话”。  
观众情绪目标：让观众感到澜冰屿真正听懂了小夏的痛点。  
画面动作：澜冰屿看向小夏、儿童鞋、隔离门、广播设备。澪星尾灯扫过地面，断裂光痕从鞋延伸到广播方向。  
关键台词：  
澜冰屿：“鞋不是源头。被切断的话才是。”  
澜冰屿：“我来固定这里。你找那句话。”  
视频化建议：静态动效。  
成本等级：低。

### K10 澜冰屿作为锚点固定空间

剧情功能：澜冰屿付出代价，固定塌陷的记忆空间，为澪星找回后半句话争取时间。  
观众情绪目标：让观众意识到澜冰屿每救一个人，自己就少一点。  
画面动作：澜冰屿站在重叠空间中心，一只手按住看不见的空间裂缝，肩膀微颤。蓝色环形瞳孔亮起，外套内侧坐标布条开始褪色。澪星守在脚边，用尾灯照住他的影子。  
视频化建议：静态动效，局部图生视频可选。  
成本等级：中。

### K11 小夏听到完整留言

剧情功能：第一集最重要的情感释放点，小夏终于知道自己没有被妈妈抛弃。  
观众情绪目标：让观众破防，但不靠大哭，而靠小夏终于确认“她不是不要我”。  
画面动作：母亲完整留言被接上后，小夏反复系鞋带的动作停住。她慢慢抬头，眼睛湿润但没有大哭，低声确认。澜冰屿看着她，轻声回答。  
关键台词：  
母亲：“小夏，别动，站在那里等我——我一定会绕回来。”  
小夏：“所以……她不是不要我？”  
澜冰屿：“不是。”  
视频化建议：图生视频。  
成本等级：高。

### K12 小夏完整系好鞋带

剧情功能：用动作证明循环解除，小夏终于完整完成系鞋带。  
观众情绪目标：让观众感到释放，小夏终于不用再用等待证明自己没有乱跑。  
画面动作：小夏蹲下，按照妈妈教的方式认真把鞋带完整系好。这一次动作没有卡住。儿童鞋化成蓝灰色记忆碎片，澪星愿望舱收纳成功。  
视频化建议：静态动效。  
成本等级：低。

### K13 小夏反问澜冰屿

剧情功能：把观众对小夏的心疼转移到澜冰屿身上。  
观众情绪目标：让观众第一次明确担心澜冰屿：他是不是也没有人绕回来找他？  
画面动作：小夏抬头看澜冰屿，轻声问他。澜冰屿听到问题后眼神停住，没有回答。手指下意识摸向外套内侧坐标布条。澪星注意到这个动作。  
关键台词：  
小夏：“那你呢？”  
小夏：“有人会绕回来找你吗？”  
视频化建议：图生视频。  
成本等级：高。

### K14 澜冰屿摸到空白坐标布条

剧情功能：将澜冰屿的代价具体化。  
观众情绪目标：让观众意识到澜冰屿不是“好像有代价”，而是真的丢失了一部分回家的路。  
画面动作：澜冰屿摸向外套内侧坐标布条，发现其中一条已经变白。他试图回忆，却想不起来。短暂停顿后，他合上外套，像什么都没发生。澪星靠近一步。  
视频化建议：静态动效或后期特写。  
成本等级：低。

### K15 澪星看见 B-07 警告

剧情功能：第一集结尾钩子。澜冰屿没有看见自己的存在痕迹下降，澪星看见了。  
观众情绪目标：让观众担心澜冰屿还能撑多久，以及澪星能不能把他带回去。  
画面动作：澜冰屿低声说自己还记得路，然后转身走入雾中。旧屏幕亮起 B-07 警告。澪星停住，回头看屏幕；它没有叫住澜冰屿，而是追上去，把尾灯照得更亮一点。  
关键台词 / 后期文字：  
澜冰屿：“我还记得路。”  
澜冰屿：“至少现在还记得。”  
屏幕文字：“B-07 锚点状态：存在痕迹下降”  
视频化建议：图生视频 / 后期文字。  
成本等级：高。

## 3. A/B/C 图生视频优先级

### A 类：最值得图生视频

这些镜头承担真正的人物动作和情绪变化，优先用于第一版 V3 情感样片。

- K06 澜冰屿半蹲面对小夏  
  情感核心：小夏不是不能走，而是不敢走。  
  关键动作：澜冰屿伸手，小夏说“妈妈回来找不到我怎么办”，澜冰屿的手停住。
- K08 澪星尾灯稳住澜冰屿  
  情感核心：澪星不是工具，而是在保护澜冰屿的存在。  
  关键动作：澪星尾巴从身后绕到身前，尾灯贴地照住澜冰屿影子，澜冰屿手指慢慢放松。
- K11 小夏听到完整留言  
  情感核心：小夏终于知道妈妈不是不要她。  
  关键动作：小夏动作停住，慢慢抬头，问“她不是不要我？”，澜冰屿轻声回答“不是”。
- K13 小夏反问澜冰屿  
  情感核心：观众对小夏的心疼转移到澜冰屿身上。  
  关键动作：小夏问“有人会绕回来找你吗？”，澜冰屿眼神停住，手摸向坐标布条。
- K15 澪星看见 B-07 警告  
  情感核心：澪星发现澜冰屿正在变少，观众开始担心主线危机。  
  关键动作：澪星回头看屏幕，澜冰屿没有看见，澪星追上去把尾灯照得更亮。

### B 类：优先静态动效

这些镜头只要构图成立，配合声音、字幕、光效、裁切和转场就能成立，原则上不优先进入可灵或其他图生视频流程。

- K01 妈妈给小夏系鞋带
- K02 澪星尾灯照见儿童鞋
- K03 水面倒影里的小夏反复系鞋带
- K04 小夏的等待规则
- K05 事故当天隔离门落下
- K07 系统识别澜冰屿
- K09 澜冰屿发现源点不是鞋
- K10 澜冰屿作为锚点固定空间
- K12 小夏完整系好鞋带
- K14 澜冰屿摸到空白坐标布条

### C 类：后期文字 / 音效 / 转场完成

这些内容不交给 AI 画面直接生成，统一后期完成。

- 系统识别文字
- B-07 锚点状态提示
- 广播器噪声
- 母亲留言被切断
- 记忆碎片收纳提示
- 屏幕 UI 闪烁
- 低频系统音
- 雨声、车站空响、远处广播残响

## 4. 第一版 V3 样片建议

第一版 V3 情感样片不做完整 17 分钟。建议先做 60-90 秒测试段，镜头顺序为：

```text
K06 → K08 → K11 → K13 → K15
```

样片内容：

```text
第一次救援失败 → 澪星稳住澜冰屿 → 小夏知道妈妈没有不要她 → 小夏反问澜冰屿 → 澪星看见 B-07
```

这段样片的目的：

- 验证观众能否共情。
- 验证澜冰屿是否有吸引力。
- 验证澪星是否有情感作用。
- 验证 V3 情感方向是否成立。
- 验证克制表演是否比单纯异常设定更打动人。
- 控制图生视频成本，不再每个镜头都反复试错。
- 用最少镜头证明第一集真正的情感骨架。

第一版样片暂不追求完整剧情解释，而是优先验证：

```text
观众能不能心疼小夏。
观众能不能开始担心澜冰屿。
观众能不能记住澪星不是工具，而是在保护他。
```

旧版 S18-S24 样片流程暂缓，不再直接作为当前制作样片。S18-S24 可保留为历史参考，但当前 P0 应切换为 K06 → K08 → K11 → K13 → K15 的 V3 情感样片。

## 5. 成本控制原则

为了避免继续陷入“每个关键帧都烧积分反复试错”的问题，V3 采用分级制作。

原则：

- A 类镜头每个最多图生视频试 2 次。
- 2 次仍不稳定，立即降级为静态动效。
- B 类镜头原则上不进图生视频。
- C 类镜头全部由后期文字、音效和转场完成。
- 不再把每个关键帧都当成图生视频任务。
- 先做 60-90 秒情感样片，不做完整 17 分钟。
- 样片通过后，再扩展全片镜头。
- 每个视频镜头生成前，必须先确认角色造型、场景、动作和情绪目标是否统一。
- 若角色脸、澪星造型或旧车站空间漂移，优先回到“序列母版关键帧”阶段，而不是继续烧视频积分。

A 类成本上限建议：

```text
K06 最多 2 次
K08 最多 2 次
K11 最多 2 次
K13 最多 2 次
K15 最多 2 次
```

如果每 5 秒约 5 元，则第一版 A 类样片理论上限约 50 元左右。实际制作时应先生成关键帧并审图，通过后再进视频，避免用视频阶段修图。

## 6. 第一版样片前置条件

进入 K06/K08/K11/K13/K15 图生视频前，必须先完成：

1. 第一集旧车站段落序列母版关键帧。
2. 澜冰屿本段造型锁定。
3. 澪星本段造型锁定，尤其是尾灯固定在尾巴最末端。
4. 小夏本段造型锁定。
5. 旧车站主空间锁定。
6. K06/K08/K11/K13/K15 五张首帧关键帧统一审阅。
7. 后期文字清单确认：系统识别文字和 B-07 文字全部后期添加，不交给 AI 生成。

## 7. 当前结论

第一集 V3 已从“事件型关键帧计划”转为“情感型关键帧计划”。当前不再继续旧版 S18-S24 样片流程。

下一步应审阅本文档，并确认：

```text
1. K01-K15 是否足以支撑第一集 V3 情感线。
2. K06/K08/K11/K13/K15 是否适合作为第一版 V3 情感样片。
3. 是否先制作旧车站段落序列母版关键帧。
4. 是否重新规划 K06/K08/K11/K13/K15 的首帧 prompt。
```
