# 《边界回声》第一集 OP03-OP16 视频台词时间槽参考 V1

用途：给剪辑、配音、口型同步和分镜核对使用。

重要规则：本文件里的“具体台词”不要复制进可灵图生视频 Prompt。给可灵的 Prompt 仍使用“台词时间槽 / 后期同步空间 / 不生成口型”的版本，避免画面生成字幕、乱口型或乱码文字。

## OP03 妈妈蹲下整理鞋子

首帧：`keyframes/op01_op12_v2/镜头03_妈妈整理小夏鞋子_v2.png`

建议时长：7 秒

台词参考：

- 1.0-2.2 秒，妈妈台词时间槽 1：`鞋松了，妈妈给你按好。`
- 2.2-3.2 秒，小夏台词时间槽 1：`我不是故意停下的。`
- 3.2-5.0 秒，妈妈台词时间槽 2：`没关系，走路慢一点也可以，鞋要穿稳。`
- 5.0-6.0 秒，小夏短反应：`嗯。`

可灵 Prompt 仍然只写：妈妈低头看鞋、小夏低头看妈妈的手、留出后期短句同步空间，不写上面这些台词。

## OP10 自动隔离栏升起

建议首帧：`keyframes/op01_op12_v2/镜头10_高长隔离门_v2.png`

建议时长：6-7 秒

台词参考：

- 0.0-1.5 秒，机械与警报声音槽：隔离门升起、系统分区音、人群挤压。
- 1.5-3.0 秒，小夏喊妈妈：`妈妈！`

注意：分离原因是人流、系统分区和高长隔离门，不是妈妈主动松手。

## OP12 妈妈留下半句话，声音被切断

建议首帧：`keyframes/op01_op12_v2/镜头12_回候车区等切断_右脚白袜_v2.png`

建议时长：7-8 秒

台词参考：

- 0.0-1.2 秒，妈妈呼喊：`小夏！回候车区！`
- 1.2-2.0 秒，小夏哭喊：`妈妈！`
- 2.0-3.5 秒，妈妈被切断的指令：`回候车区等——`
- 3.5-4.5 秒，被灾难切断但后续剧情要找回的后半句：`妈妈一定会来接你！`
- 4.5-8.0 秒，小夏实际只听见前半句，声音戛然而止，画面落到掉落的儿童鞋。

注意：正片观众在 OP12 可以只明确听到 `回候车区等——`，后半句可以被爆音和隔离门切掉，留到后续调查补全。

## 统一复制规则

给可灵复制 Prompt 时，只复制原始视频 Prompt 中的画面、动作、台词时间槽、限制和负面限制。

不要复制本文件中的具体台词到可灵 Prompt。

本文件只用于你自己确认“这个时间槽里人物说的是什么”。
