- 所有继续生成的视频和补帧，必须以 `assets/characters` 内角色海报 V2、`assets/characters/陆衡_V3发型候选/陆衡_V3候选B_自然分束湿发_透明版.png` 和 `ep02_director_v3` 为参考。
- 澜冰屿必须是湿乱垂落、病弱内收剪影；陆衡必须是 V3 选定自然分束湿发、露眉眼、露耳机、外放压迫调查员剪影。
- 澪星必须是小型机械猫形引航生命，不能生成真实动物、毛绒宠物、鸟类或怪物。
- 屏幕类镜头不要生成可读长文字，只保留抽象界面和短标识。
- 陆星遥不要完整实体登场，继续保持断裂投影和残音状态。
