# 角色锁整理

## 澜冰屿

### 角色定位

男一号，人形锚点，异常区长期幸存者，记忆回声感知者。

### 外形固定特征

苍白肤色，极细蓝色环形瞳孔，短黑发为主，黑发层里零碎散落少量白色发丝，白色部分是细碎、断续、随机分布的发丝，不形成一束一束的挑染，安静、迟缓、像被世界遗忘。

### 服装固定特征

米白色破旧长外套，浅灰高领内搭，旧黑裤，磨损短靴，外套内侧缝着手写坐标布条。

### 气质

克制、安静、诚实、疲惫、温柔但不外露，像比世界慢半秒。

### 画风统一约束

澜冰屿进入剧情关键帧时必须跟小夏、妈妈所在的事故段保持同一套高精度写实 CG 渲染风 / 电影级关键帧插画风。人物像剧中真实存在的人，不像动画男主、漫画美少年、游戏角色立绘或角色海报站桩。生成关键帧时不要使用“animated series”“anime concept art”“正式动画剧集剧情帧”等正向风格词；只保留真实物理光照、电影级 CG 光影、克制表演、真实布料磨损和低饱和冷灰蓝雨夜环境。

### 禁止漂移项

不要变成长发美少年，不要偶像化，不要战斗英雄姿势，不要黑色风衣替代白旧外套，不要去掉坐标布条，不要换瞳色，不要变真人短剧男主，不要把白发画成一束一束的挑染、半头白、大片银白刘海或全白发。

### 视频生成时的稳定写法

```text
澜冰屿保持同一张脸、同一年龄感、苍白肤色、短黑发为主且黑发层里零碎散落少量白色发丝、极细蓝色环形瞳孔，穿米白色破旧长外套和浅灰高领，外套内侧可见手写坐标布条。白色头发必须像 V2 版海报2：细碎、断续、随机分布，不是一束一束的白色挑染。动作克制，只做轻微抬头、低头、停顿、慢步或手指轻触，不做夸张姿势。
```

## 澪星

### 角色定位

边界回声引导生命体，澜冰屿的陪伴与路径引导者，负责感知回家路和残留回声。

### 外形固定特征

蓝灰磨损外壳，奶白脸腹，青蓝眼，额头青蓝引航星核，橙色耳灯，斜挎愿望舱小包，尾部悬挂引航灯。与澜冰屿同框时，澪星高度约到澜冰屿膝盖，不是脚踝高度的小玩具，也不是掌心宠物尺寸。

### 引航灯方向锁

正常状态下，尾部引航灯的底部与尾巴相连，顶部的小星星朝上。尾巴伸到身前照亮物体时，引航灯必须倒过来：灯的底部在上方并与尾巴相连，顶部的小星星朝下。不要把照明状态画成星星朝上、底部悬空。

### 禁止漂移项

不要缩得过小，不要变成普通玩具猫、毛绒玩具或二次元宠物，不要去掉额头引航星核、橙色耳灯、愿望舱小包和尾部引航灯，不要把照明状态下的引航灯方向画反。

### 视频生成时的稳定写法

```text
澪星保持 V2 版引导生命体造型：蓝灰磨损外壳、奶白脸腹、青蓝眼、额头青蓝引航星核、橙色耳灯、斜挎愿望舱小包、尾部悬挂引航灯。与澜冰屿同框时高度约到澜冰屿膝盖。正常状态下尾灯底部与尾巴相连、星星朝上；尾巴伸到身前照亮鞋子时，尾灯倒置，底部在上方连接尾巴、星星朝下。动作以低头观察、抬头看澜冰屿、尾灯照明和轻微提示音为主。
```

## 陆衡

### 角色定位

男二号，失踪事件调查员，妹妹线，现实证据与追查视角。

### 外形固定特征

29 岁清瘦疲惫东亚男性，黑色短发略乱，眼下有疲态，右眼下浅旧伤，左耳旧式骨传导通讯器。

### 服装固定特征

短款深灰蓝调查夹克，黑色连帽内搭，单肩战术背带，旧相机包，手腕旧式取证终端，深色长裤，护膝，磨损旅行靴。

### 气质

敏锐、压抑、不轻信、专注，像一直被失踪案拖住的人。

### 禁止漂移项

不要长风衣，不要刑侦爽剧男主，不要西装精英化，不要变战斗特工，不要去掉旧相机包、取证终端和疲惫感。

### 视频生成时的稳定写法

```text
陆衡保持清瘦疲惫的 29 岁东亚调查员形象，黑色短发略乱，右眼下浅旧伤，左耳骨传导通讯器，短款深灰蓝调查夹克、黑色连帽内搭、旧相机包和手腕取证终端保持不变。动作以翻档案、抬眼观察、停顿、手指触碰终端为主。
```

## 许明烬

### 角色定位

女二号，前城市轨道维修师，废城跑单人，黑市维修师，现实行动与设备修复担当。

### 外形固定特征

29 岁东亚女性，短狼尾，后颈碎发，刘海不整齐，一侧金属发夹或细扎带，嘴角或眉骨旧伤。

### 服装固定特征

褪色棕红短款维修夹克，深色工字背心或贴身机能内搭，黑色工装裤，旧手套，磨损靴，腰间维修钥匙串、胶带、短工具、旧轨道工牌，右手低调工业机械手，指节暖橙状态灯。

### 气质

嘴硬、行动派、讲义气、带旧伤和赎罪感，轻松语气下面压着事故记忆。

### 禁止漂移项

不要改成长发女战士，不要重甲机甲，不要性感化，不要超级英雄机械臂，不要高饱和红皮衣，不要去掉维修生活痕迹。

### 视频生成时的稳定写法

```text
许明烬保持 29 岁东亚维修师形象，短狼尾、金属发夹、褪色棕红短款维修夹克、深色工字背心、黑色工装裤、低调工业机械右手和暖橙指节状态灯保持一致。动作以接电、检查接口、短暂停顿、听到广播后眼神变化为主。
```

## 林见舟

### 角色定位

女一号，28 岁，静海断层后的记忆治疗师和情绪调律师，主角团的情绪核心。

### 外形固定特征

成熟温柔的东亚女性，中短深棕发，发尾有极细的玫瑰金光纤般微光，温柔但坚定的眼神，气质清醒、坚韧、有专业感但不冷漠。

### 服装固定特征

雾白色长外套，浅茶色内搭，低饱和玫瑰色不对称围巾，胸前嵌着一枚半透明心律共鸣晶体。晶体是静海系统灾后心理修复项目里的医疗调律设备，不是魔法宝石。

### 气质

温柔、清醒、坚韧，能陪别人把最痛的那一刻看完，但不会轻易承诺“会好起来”。她是最早坚持澜冰屿“不是故障，而是一个人”的人。

### 禁止漂移项

不要写成林间舟，不要改成路线读取者，不要改成法师、占星师、仙气古装女主、偶像女主或治愈系圣女，不要去掉雾白长外套、玫瑰围巾、心律共鸣晶体和专业克制感。

### 视频生成时的稳定写法

```text
林见舟保持 28 岁成熟温柔的东亚记忆治疗师形象，中短深棕发、发尾极细玫瑰金光纤微光、雾白色长外套、浅茶色内搭、低饱和玫瑰色不对称围巾、胸前半透明心律共鸣晶体保持一致。动作以手指靠近心律晶体、低声引导、停顿倾听、稳定情绪污染为主。她的能力是医疗调律与共情锚定，不是魔法。
```

## 泊舟

### 角色定位

泊舟不是已完整确认的人形角色，而是静海系统灾后引导协议和记忆舱 / 引航体系的视觉对象。当前应按“AI 记忆舱、停泊点、引航装置、系统残留单位”保守使用。

### 外形固定特征

低饱和旧白与雾蓝外壳，圆润但不玩具化，像旧系统中的小型记忆舱或停泊舱，表面有细微磨损、低亮度信标、冷白淡蓝状态光。

### 服装固定特征

不适用。泊舟作为装置时应保留旧系统外壳、记忆舱接口、微弱引航灯和静海系统残留标记，但不出现可读 logo。

### 气质

安静、旧、可靠、略带哀伤，像曾经用于救援但已经被废城遗忘的装置。

### 禁止漂移项

不要做人形 AI 偶像，不要飞船机甲化，不要高饱和科幻飞行器，不要可爱玩具化，不要强烈赛博霓虹。

### 视频生成时的稳定写法

```text
泊舟表现为静海系统残留的旧式 AI 记忆舱 / 引航停泊装置，低饱和旧白和雾蓝外壳，微弱冷白淡蓝状态光，表面有细小磨损和旧金属接缝。动作只允许轻微悬浮、低频闪烁、舱体缓慢呼吸式发光，不变形成机甲或飞船。
```

## 洛衡

### 角色定位

洛衡当前缺少完整角色圣经记录，暂作为“读取旧路线的导航者 / 路线分析者”占位整理。后续如确认真实身份，应回写角色圣经。

### 外形固定特征

成熟克制的东亚角色，年龄感稳定，不低龄化。外形应服务“读取旧路线、观察废城、判断方向”的功能，不抢主角团既有识别点。

### 服装固定特征

低饱和深灰蓝或旧白机能外套，简洁路线记录终端，少量暗金或冷白导航光。不要使用与陆衡、澜冰屿高度重复的核心服装特征。

### 气质

冷静、观察型、像熟悉旧路线和异常区边界的人，情绪克制，不神秘卖弄。

### 禁止漂移项

不要改成奇幻占星师，不要魔法师，不要偶像化，不要高饱和星空斗篷，不要与现有角色撞脸撞服装。

### 视频生成时的稳定写法

```text
洛衡保持成熟克制的路线读取者形象，低饱和深灰蓝或旧白机能外套，手持简洁旧路线终端，站在高处读取废城旧路线。动作以低头查看终端、抬眼望向远处轨道、衣摆被风轻动为主。不要魔法化，不要高饱和星空特效。
```
