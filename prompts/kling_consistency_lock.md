# 全局视频一致性锁 Prompt（可灵用）

## 固定画风锁

```text
全片保持成熟向半写实科幻动画剧集风格，像正式动画剧集中的剧情镜头，不是真人短剧，不是游戏 CG，不是角色海报。画面真实但不照片化，冷灰蓝废城基调，低饱和，电影级镜头语言，克制情绪，静默悬疑，微弱记忆光。
```

## 固定角色锁

```text
保持同一角色脸型、年龄感、发型、服装、配件和气质，不要换脸，不要改变发型，不要改变服装颜色，不要让角色变成真人演员或另一个动画风格。角色动作小而稳定，情绪通过眼神、呼吸、停顿和手部小动作表达。
```

## 固定服装锁

```text
服装必须延续上一镜头：外套长度、颜色、磨损、内搭、靴子、配件位置保持一致。不要自动添加新披风、新盔甲、新项链、新制服、新 logo 或现代潮牌元素。
```

## 固定场景锁

```text
同一段落保持同一场景体系：建筑材质、地面湿度、雾气浓度、旧屏幕位置、金属设施、时间段和天气连续稳定。不要从旧车站突然变成现代商业街、霓虹城市、游戏副本或空旷摄影棚。
```

## 固定色彩锁

```text
全段使用冷灰蓝、旧白、深灰黑、铁锈红和少量暗金。暖橙只作为旧设备或机械状态灯，淡蓝冷白只作为记忆投影和引航光。不要高饱和霓虹，不要糖果蓝紫，不要大面积粉色梦幻滤镜。
```

## 固定光照锁

```text
保持冷雾、雨后反光、柔和体积光和低亮度定向光。不要突然切换成阳光棚拍、商业广告光、强烈舞台光或过曝逆光。光源变化要缓慢、合理、服务叙事。
```

## 固定镜头语气锁

```text
镜头语气克制，像悬疑科幻动画剧集。推镜、拉镜、横移、低机位跟随、局部光效、轻微雾层流动可以使用。不要高速摇镜，不要夸张旋转，不要战斗特效，不要 MV 式炫技剪辑。
```

## 固定时间段锁

常用时间段请明确写入每条视频 prompt：

- 夜雨：旧车站、地下轨道、系统压迫段。
- 黄昏：废城旧车站、泊舟待机、群像停留。
- 清晨冷雾：异常区边缘、回收遗憾后的余韵。
- 深夜冷光：档案室、控制间、失踪线索调查。

## 固定动作强度建议

适合可灵稳定生成：

- 轻微抬头
- 缓慢转身
- 低头查看物件
- 风吹衣摆
- 慢步前进
- 手指轻碰物件
- 轻微呼吸起伏
- 视线偏移
- 短暂停顿
- 尾灯轻微呼吸
- 屏幕低频闪烁

避免：

- 跑跳打斗
- 大幅挥手
- 快速转身
- 大量人物穿插
- 复杂手指操作
- 强烈表情变化
- 镜头快速旋转

## 固定表情控制建议

```text
表情克制，不夸张哭喊，不中二怒吼。澜冰屿安静、疲惫、像比世界慢半秒；陆衡警觉、压抑、专注；许明烬嘴硬但眼神有旧伤；澪星温柔、警觉、陪伴；林见舟温柔清醒。
```

## 视频段落之间承接写法

```text
延续上一镜头的同一场景、同一时间、同一天气和同一色彩体系。角色服装、发型、配件、年龄感和脸型保持一致。镜头从上一镜头的动作余韵开始，动作幅度小，画面只发生轻微推进、视线变化、雾气流动和光线变化。
```

## “上一镜头延续”写法模板

```text
承接上一镜头：保持同一废弃旧车站、夜雨、冷灰蓝低饱和色调、潮湿地面反光和冷雾。角色保持上一镜头的服装、发型、脸型、年龄感和位置关系。镜头从上一镜头结束后的短暂停顿开始，只做轻微推镜和呼吸式光线变化，不改变画风，不改变角色比例，不新增人物。
```
