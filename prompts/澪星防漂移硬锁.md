# 澪星防漂移硬锁

## 必须使用的正向描述

澪星必须严格参考：

- `F:\CodexSharedCockpit\assets\characters\澪星_角色海报_v2.png`
- `F:\CodexSharedCockpit\assets\characters\澪星_角色海报_v2_设定板.png`

稳定写法：

```text
澪星是 V2 设定板同款膝高圆润机械引航生命，不是动物。蓝灰磨损硬质金属外壳，奶白脸腹，青蓝玻璃质大眼，额头青蓝引航星核，橙色圆形耳部信号灯，斜挎愿望舱小包，尾部悬挂蓝光引航灯。身体是硬质机械外壳和金属接缝，不是毛发、皮肤或毛绒玩具。与澜冰屿同框时高度约到澜冰屿膝盖。
```

## 禁止使用的误导正向词

以下词可以在解释里出现，但不要作为图像生成的正向主描述：

- 机械猫形
- 小型机械猫
- 猫形身体
- 可爱猫
- 宠物
- 玩具
- 吉祥物

这些词会让模型把澪星生成成真实猫、玩具猫、毛绒宠物或儿童动画角色。

## 必须追加的负面限制

```text
不要真实猫，不要普通猫，不要动物，不要动物毛发，不要猫爪肉垫，不要真实猫鼻纹理，不要真实猫胡须，不要毛绒玩具，不要儿童玩具，不要二次元宠物，不要哆啦A梦，不要机器狗，不要怪物，不要吉祥物玩偶。不要去掉额头引航星核、橙色耳部信号灯、愿望舱小包和尾部引航灯。
```

## 判断标准

合格澪星：

- 一眼是 V2 设定板里的机械引航生命。
- 有硬质蓝灰外壳和金属磨损。
- 有额头星核、橙色耳部信号灯、愿望舱小包、尾部引航灯。
- 尺寸到澜冰屿膝盖附近。

失败澪星：

- 像真实猫、玩具猫、毛绒猫、宠物、儿童动画吉祥物。
- 出现动物毛发、猫爪肉垫、猫鼻、猫胡须。
- 没有星核、耳部信号灯、愿望舱或尾灯。
