# 全局风格锁 Prompt

## 全局风格正向提示词

成熟向半写实科幻动画剧集风格，像高质量科幻动画剧集中的剧情帧，不是真人短剧，不是游戏设定图。冷灰蓝废城基调，真实但不照片化的动画光影，电影级构图，克制情绪，静默悬疑，废弃城市、异常区、旧车站、地下轨道、档案室、雨后工业遗迹。真实世界残骸与异常区异质感并存，微弱记忆光，低饱和冷灰蓝、旧白、深灰黑、铁锈红、少量暗金和暖橙状态灯。角色像剧中人物，表情克制，情绪通过眼神、姿态和小动作表达。镜头有明确景别、叙事目的和环境关系。

## 全局负面提示词

不要真人短剧风，不要短视频网剧风，不要低龄玩具感，不要儿童卡通感，不要纯二次元萌系风，不要夸张网游设定图风，不要海报站桩风，不要过度华丽的赛博朋克霓虹，不要过度梦幻星光滤镜，不要照片级写实，不要时尚杂志感，不要游戏角色立绘，不要三视图设定页，不要超级英雄感，不要机甲战斗爽感，不要廉价 AI 感，不要面部崩坏，不要手指畸形，不要过度锐化，不要角色年龄漂移，不要服装随意变化，不要发型随机变化，不要人物比例漂移，不要场景时代感混乱，不要任何现成知名角色特征，不要可读乱码文字，不要水印，不要 logo。

## 全局镜头 / 色彩 / 光线 / 材质 / 情绪约束

镜头必须像动画剧集中的剧情镜头，有明确景别和叙事情境。优先中景、近景、半身、大全景环境关系镜头。画面以冷灰蓝和旧白为底，铁锈红和暗金只作辅助，暖橙只用于旧设备或机械状态灯。光线为电影感定向光和柔和体积光，雨后地面反光、旧金属反射、灰尘和冷雾可以出现。材质保留旧世界磨损、潮湿、铁锈、旧布料、低亮度屏幕和微弱记忆投影。情绪必须克制、安静、悬疑、孤独，但保留人性温度。
