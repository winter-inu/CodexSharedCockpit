# 全局风格锁 Prompt

## 全局风格正向提示词

高精度写实 CG 渲染风 / 电影级关键帧插画风，看起来像真人电影被按下了暂停键，但明确不是真人拍摄。真实物理光照，高对比，强氛围感，电影级构图，克制情绪，静默悬疑。皮肤有真实肌理但没有毛孔级照片写实，避免恐怖谷；眼神克制、有重量，不二次元空洞。废城、雨、雾、旧车站、地下轨道、档案室、雨后工业遗迹像布景精美的实景片场。动作停在高情感浓度的一瞬，不追求流畅动画感。低饱和冷灰蓝、旧白、深灰黑、铁锈红、少量暗金和暖橙状态灯；事故前段可使用午后暖光和生活感。角色像剧中人物，情绪通过眼神、姿态和小动作表达。镜头有明确景别、叙事目的和环境关系。

## 全局镜头 / 色彩 / 光线 / 材质 / 情绪约束

镜头必须像电影级关键帧中的剧情镜头，有明确景别和叙事情境。优先中景、近景、半身、大全景环境关系镜头。画面以低饱和冷灰蓝和旧白为底，铁锈红和暗金只作辅助，暖橙只用于旧设备或机械状态灯；OP01-OP06 允许午后暖光和生活感。光线为真实物理光照、电影感定向光和柔和体积光，雨后地面反光、旧金属反射、灰尘和冷雾可以出现。材质保留旧世界磨损、潮湿、铁锈、旧布料、低亮度屏幕和微弱记忆投影。情绪必须克制、安静、悬疑、孤独，但保留人性温度。
