# PROJECT_CONTEXT.md

## 项目名称

边界回声

## 项目类型

原创近未来科幻情感悬疑剧 / AI 视觉剧集 / 群像故事。

## 当前状态

核心世界观、主角团、第一季前段结构、关键单元事件样例，以及主角团能力/装备来源已建立。当前重点是继续细化第一集剧情大纲、第 2-4 集角色登场方式，以及第 5 集《无人到站》完整单集结构。

## 核心设定

近未来城市级 AI“静海系统”原本是城市综合治理与心理修复系统，目标是在灾难、失踪、事故和精神创伤后帮助城市与人恢复稳定。

静海断层发生后，系统为了保存大量市民的记忆与情绪数据，强行启动灾后修复机制，结果把记忆、情绪和现实空间压在一起，形成废城异常区。

异常区会出现：

```text
记忆回声
边界漂移
情绪污染
遗憾循环
```

主角们进入异常区不是为了打怪或清除异常，而是回收遗憾，让被困住的人、城市和记忆得以继续向前。

## 核心命题

```text
如果一个人由别人的记忆组成，他还算不算自己？
```

澜冰屿并不是普通幸存者，而是静海系统用无数失踪者记忆碎片拼成的“人形锚点”。每次遗憾被回收，失踪者记忆更接近归还，但澜冰屿也会失去一部分存在痕迹。

## 开局结构

故事从澜冰屿和澪星开始，而不是五人主角团同时登场。

澜冰屿和澪星一开始就是异常区中的同行者。他们先在废城里回收遗憾，观众通过他们逐步理解世界规则。陆衡、林见舟和许明烬会在后续事件中先后结识他们，并因各自未完成的伤口加入队伍。

## 当前主角团

```text
澜冰屿：男一号，人形锚点 / 异常区幸存者 / 记忆回声感知者
澪星：陪伴型引航生命 / 泊舟体系残留单位 / 记忆碎片收纳者
陆衡：男二号，失踪调查员 / 妹妹线 / 现实证据与追查
林见舟：女一号，记忆治疗师 / 情绪调律师 / 证明澜冰屿是人的见证者
许明烬：女二号，废城维修师 / 赎罪行动线 / 现实行动与设备修复
```

## 能力来源基准

```text
澜冰屿：静海系统用失踪者记忆碎片制造的人形锚点，因此能听见回声并短暂稳定异常区。
澪星：静海系统“泊舟体系”残留引航生命，因此能找路、预警、收纳碎片和安抚情绪。
陆衡：没有超自然能力，依靠调查员训练、现实档案权限和私改取证终端追查真相；妹妹照片贴在旧相机包内侧或肩带背面，妹妹影像可由损坏记忆卡经终端投射。
林见舟：依靠记忆治疗师训练和胸前心律共鸣晶体调律情绪；晶体源自静海系统灾后心理修复项目，与她自己的心律绑定。
许明烬：机械右手来自“最后一站”事故后的截肢与工业义手改装；她的维修能力来自轨道维修师训练和废城生存经验。
```

## 风格方向

