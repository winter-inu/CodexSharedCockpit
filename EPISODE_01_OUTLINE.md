# EPISODE_01_OUTLINE.md

## 版本状态

V2。第一集《尾灯照见的鞋》已从短片式回收大纲升级为约 15 分钟单集剧情结构。

本版保留 V1 的核心事件：小夏、儿童鞋、等待母亲、被切断的留言、澜冰屿付出存在痕迹代价，并新增第一次失败段落、三层阻力和更完整的单集节奏。

## 本集标题

第 1 集：《尾灯照见的鞋》

## 建议时长

约 15 分钟。

## 类型

原创近未来科幻情感悬疑单集。

## 本集功能

这一集是《边界回声》的观众入口，负责建立：

1. 澜冰屿与澪星的同行关系。
2. 异常区不是打怪空间，而是被遗憾困住的记忆现实混合地带。
3. 遗憾回收的基本规则：找到真正源点，把被切断、被误解、没能抵达的话还给当事人。
4. 澜冰屿不是普通人，他能作为锚点稳定异常区。
5. 每一次回收遗憾，澜冰屿都会失去一点存在痕迹。
6. `B-07` 主线伏笔。

本集只以澜冰屿和澪星为核心。陆衡、林见舟、许明烬不正式登场、不入队。

## 核心结构

```text
发现异常
→ 追踪源点
→ 进入遗憾循环
→ 第一次失败
→ 找到真正源点
→ 澜冰屿作为锚点
→ 回收遗憾
→ B-07 结尾钩子
```

## 三层阻力

```text
空间阻力：车站、临时安置点、失物招领室不断切换，路径不稳定。
系统阻力：广播、闸机、登记屏和识别机制不断要求“留在原地等待确认”。
人物阻力：小夏一开始相信母亲会回来，后来被切断留言反复伤害，逐渐相信自己被忘记。
```

## 0:00-1:30 冷启动：废弃车站里的尾灯

深夜，静海断层后的废弃车站被雨声、冷雾和破损广播包围。站台电子屏已经腐蚀成乱码，只有“临时安置点请向左”的旧指示牌偶尔闪一下。

广播断断续续播放：

```text
请看管好您的随身物品和同行儿童。
```

雾里先出现澪星的青蓝尾灯。它从轨道边缘慢慢走到站台黄线旁，尾灯照亮一只湿漉漉的儿童鞋。鞋带散开，鞋底沾着灰黑泥水。

澪星没有立刻碰鞋，而是回头看向雾里。

澜冰屿从雾中出现，米白色旧长外套被雨雾打湿，外套内侧隐约露出手写坐标布条。他看着那只鞋，像听见了很远的声音。

他说：

```text
又有人没回家。
```

## 1:30-3:00 发现异常：这不是普通遗失物

澜冰屿靠近儿童鞋时，鞋子下方的积水倒影发生变化：倒影里出现一个蹲着系鞋带的小女孩。现实中站台空无一人。

澪星用额头引航星核标记鞋子的记忆坐标，愿望舱包轻轻震动，却无法直接收纳它。澜冰屿判断：

```text
不是碎片，是入口。
```

他听见三层声音重叠在一起：

```text
一个孩子问：妈妈是不是去登记处找我了？
一个女人在很远的广播里说：别动，站在那里等我。
一个系统提示音不断重复：未识别监护人，等待确认。
```

车站雾气变浓，电子屏刷新出一条残缺安置登记信息：

```text
儿童临时安置：等待监护人确认。
```

澜冰屿意识到：小女孩不是在找鞋，而是在等一句没能完整抵达的话。

他说：

```text
她不是在找鞋，她在等一句话。
```

## 3:00-5:00 追踪源点：失物招领室

澜冰屿和澪星沿着尾灯照出的路径进入车站内部。站台、事故当天的临时安置点、失物招领室三种空间不断切换。

小型异常开始出现：

```text
一排储物柜自动弹开，里面只有潮湿的旧登记单。
一个儿童水壶滚到澜冰屿脚边，壶里传出压抑的哭声。
广播忽然改口，温和地诱导澜冰屿走向错误通道。
```

澜冰屿一度被广播牵引，走向一条看似安全、实则不断重复的白色走廊。澪星没有出声，只是把尾灯压低，横在他脚边，把他的方向拉回失物招领室。

失物招领室门口堆满旧伞、学生卡、儿童水壶和无人认领的包。门牌在“失物招领室”和“临时安置点”之间闪烁。

门缝下有一只小手的影子反复推门，像里面有人一直想出来，却总是被推回去。

## 5:00-7:30 进入遗憾循环：事故当天

澜冰屿把手贴上门。门后不是房间，而是一段被保存过度的遗憾循环。

车站恢复到静海断层当天。人群拥挤，广播混乱，隔离门一扇扇落下。小夏第一次完整出现：约 6-8 岁，抱着小布包，一只鞋带松开。

她和母亲被人流与隔离门分开。母亲在另一侧拍门，拼命喊：

```text
小夏，别动，站在那里等我。我一定会绕回来。
```

但广播在中途切断母亲后半句，只留下冷冰冰的系统提示：

```text
请儿童留在原地。
监护人确认失败。
请儿童留在原地。
```

小夏一开始相信母亲会回来。她抱着小布包，站在儿童等待区边缘，一遍遍看向母亲消失的方向。

循环重复得越久，她越只听见“留在原地”，越听不见“我一定会绕回来”。她从相信母亲会回来，逐渐怀疑自己被忘记。

## 7:30-9:30 第一次失败：澜冰屿不能直接救她

澜冰屿试图带小夏离开儿童等待区。他没有解释太多，只伸手示意她跟着尾灯走。

但循环立刻把澜冰屿识别成“未登记人员”。

闸机、隔离栏、电子屏同时转向他。失物招领室门牌熄灭，临时安置点指示牌全部指向同一个原地等待区。

系统提示音压下来：

```text
请留在原地。
请等待确认。
请留在原地。
```

澜冰屿脚下的影子变浅。他差点被拖入“等待者”身份，像小夏一样被登记、被确认、被留在原地。

澪星迅速靠近，用尾灯照住他的脚边。青蓝光带把他的影子重新压回地面。

澜冰屿低声说：

```text
我还在。
```

第一次尝试失败。澜冰屿意识到，不能直接把小夏带走。只要那句话仍然被切断，小夏就会被循环推回等待的起点。

## 9:30-11:30 找到真正源点：不是鞋，是被切断的留言

澜冰屿开始观察每一次循环的崩溃点。

小夏并不是在鞋掉落时崩溃，也不是在母亲被隔离门挡住时崩溃。真正让她被困住的，是广播切断母亲留言的那一秒。

每一次，小夏都只能听见：

```text
别动，站在那里等我。
```

她听不到后半句：

```text
我一定会绕回来。
```

真正的遗憾不是鞋，而是小夏只听到了等待，没听到母亲会回来。

澜冰屿看向澪星，说：

```text
我来固定这里。
你找那句话。
```

澪星尾灯亮度降低，像一盏贴近地面的航标。它沿着失物招领室、广播喇叭、安置登记屏之间的断裂光痕寻找那句被切掉的话。

## 11:30-13:00 高潮：澜冰屿作为锚点

车站空间开始塌陷。候车区、失物招领室、临时安置点重叠在同一条走廊里。小夏一遍遍被推回蹲下系鞋带的动作。

澜冰屿站在中央，作为人形锚点稳定循环。

他的蓝色环形瞳孔亮起。脚下的空间裂纹被短暂压住。外套内侧的手写坐标布条开始褪色，一条布条的字迹从边缘被擦去。

澪星用尾灯照住他的脚边，确认他的影子没有被吞掉。

澜冰屿走到小夏面前，蹲下，把儿童鞋推回她够得到的位置。

他说：

```text
你不是被忘记了。
```

澪星找到那句被切断的留言。断裂的蓝灰色记忆波纹重新接上。

澜冰屿把完整留言还给小夏：

```text
她说的是——我一定会绕回来。
```

## 13:00-14:20 回收遗憾：鞋化为记忆碎片

小夏终于不再盯着冰冷的登记屏。她低头系好鞋带，第一次主动离开黄线。

儿童鞋化成蓝灰色记忆碎片和细小星屑，缓慢落入澪星的愿望舱包。失物招领室和临时安置点的重叠空间开始安静下来。

小夏的身影不再重复等待动作。她看着澜冰屿，问：

```text
那你呢？
有人会绕回来找你吗？
```

澜冰屿没有回答。

他像是听懂了这个问题，又像是不知道自己有没有资格被这样问。

## 14:20-15:30 结尾钩子：B-07

回收完成后，澜冰屿站在原地，摸向外套内侧。

一条手写坐标布条已经完全变白。

他知道那里原本写着某个人最后出现的位置，却想不起具体地址，也想不起那个人是谁。

澪星靠近他，用额头星核轻轻碰了碰他的手背。等澜冰屿把手放下来后，澪星又用尾灯照了一下他的脚边，确认他的影子没有继续变淡。

澜冰屿把空白布条重新缝回外套内侧。

他说：

```text
走吧。还有人没回家。
```

清晨前，澜冰屿和澪星离开废弃车站。澪星的愿望舱包里，那枚儿童鞋碎片已经安静下来。

在澜冰屿转身离开后，失物招领窗口后的旧屏幕短暂亮起：

```text
B-07 锚点状态：存在痕迹下降
```

澜冰屿没有看见。

澪星看见了。它的尾灯亮度短暂升高，又很快压低。它停在原地回头，确认澜冰屿还在，才追上去。
