# EPISODE_01_OUTLINE.md

## 版本状态

V1 草案。基于 V0 小修，保留核心事件，弱化与第 5 集《无人到站》可能重复的“末班车 / 已到站 / 无人列车”意象，第一集重点改为“失物招领室 / 临时安置点 / 儿童鞋 / 等待被误解”。

## 本集标题

第 1 集：《尾灯照见的鞋》

## 本集功能

这一集是《边界回声》的观众入口，只建立最必要的世界规则和人物关系。

本集必须让观众理解三件事：

1. 澜冰屿和澪星不是在打怪，而是在帮助被困住的遗憾继续向前。
2. 澜冰屿不是普通人，他能听见记忆回声，并能短暂稳定异常区。
3. 每一次帮助别人，澜冰屿都会失去一点自己的存在痕迹。

本集只以澜冰屿和澪星为核心。陆衡、林见舟、许明烬不正式登场、不入队。

## 开场冷启动

深夜，静海断层后的废弃车站被雾包住。站台电子屏忽明忽暗，班次信息已经腐蚀成乱码，只剩“临时安置点请向左”的旧指示牌还在偶尔发光。广播反复播放一段破损提示音：

```text
请看管好您的随身物品和同行儿童。
```

雾里先出现一盏柔和青蓝色尾灯。澪星从轨道边缘慢慢走出，尾灯拖出很短的光带。它停在站台黄线旁，额头引航星核微亮，低头看见一只湿漉漉的儿童鞋。

鞋很小，鞋带散开，鞋底沾着灰黑色泥水。它不该出现在已经废弃多年的站台上。

澪星没有立刻碰它，而是回头望向雾里。澜冰屿从雾中走出，米白色旧长外套边缘被雨水打湿，外套内侧隐约露出几条手写坐标布条。

他看着那只儿童鞋，像听见了很远的声音，轻声问：

```text
找到了吗？
```

澪星尾灯亮了一下。车站广播忽然切成童声哼唱，站台深处出现一个模糊的小女孩影子，蹲在失物招领窗口旁系鞋带。下一秒，灯光闪断，影子消失，只剩那只鞋还在原地。

## A段：发现异常

澜冰屿靠近儿童鞋时，站台空间开始轻微错位：候车椅多出一排，失物招领窗口从生锈状态短暂恢复成事故当天的样子，墙上的临时安置点指示牌浮现出一串登记编号。

澪星用额头星核标记儿童鞋的记忆坐标，愿望舱包轻轻震动，却无法直接收纳它。说明这不是普通记忆碎片，而是遗憾循环的入口。

澜冰屿听见三层重叠回声：

```text
一个孩子问：妈妈是不是去登记处找我了？
一个女人在很远的广播里说：别动，站在那里等我。
一个系统提示音不断重复：未识别监护人，等待确认。
```

车站雾气突然变浓，站台电子屏刷新出一条残缺的安置登记信息：

```text
儿童临时安置：等待监护人确认。
```

澜冰屿判断这里的遗憾源点不是“孩子丢了鞋”，而是“孩子一直以为自己被留在了原地”。澪星尾灯指向站台尽头的失物招领室，提示源点藏在“等待登记”和“无人认领”的夹缝里。

## B段：追踪遗憾源点

澜冰屿和澪星沿着尾灯照出的安全路径进入车站内部。边界开始漂移，站台、扶梯、候车厅和失物招领室反复改变位置。澪星每走几步就停下，用尾灯重新校准方向，避免澜冰屿被错误通道带入更深的循环。

他们在候车厅和临时安置通道之间看到事故当天的记忆回声：

```text
避难人群挤向闸机。
一名母亲逆着人流跑回失物招领窗口。
一个小女孩抱着布包，鞋带散开，却被广播要求到儿童等待区登记。
系统把“等待监护人确认”保留下来，却没有保存母亲后来绕回来的完整留言。
```

澜冰屿没有完整读取过去，只能接触到残留情绪和关键回声。他被孩子的恐惧牵引，短暂说出一个自己不该知道的称呼：

```text
小夏。
```

说出口后，他自己也愣住。他不知道“小夏”是谁。澪星立刻靠近，用额头星核低频闪烁，像是在提醒他保持现实感。

失物招领室门口堆满旧伞、学生卡、儿童水壶和无人认领的包。门牌在“失物招领室”和“临时安置点”之间闪烁。澪星尾灯照到门缝下，一只小手的影子正在里面反复推门。

澜冰屿把手贴上门。门后不是房间，而是一段被保存过度的记忆循环。他只能听见碎片，无法解释整个系统为什么这样保存。

## C段：进入遗憾循环

澜冰屿进入循环，澪星跟在他身后，用尾灯维持返回坐标。

循环里，车站恢复到静海断层当天。小女孩“小夏”站在儿童等待区边缘，脚上一只鞋松了。她的母亲原本已经带她走向出口，但人流和隔离门突然落下，把两人分开。

母亲在另一侧不断拍门，让她别乱跑，自己会回来接她。可广播将母亲的声音切碎，只留下冷冰冰的系统提示：

```text
请儿童留在原地。
监护人确认失败。
请儿童留在原地。
```

小夏一直等。登记屏一次次刷新“监护人确认失败”，却没有播放完整留言。她从相信母亲会回来，变成怀疑自己是不是被忘记了。这个念头太强，最终把车站变成了遗憾循环：她永远等在原地，永远少一只鞋，永远听不完整母亲最后那句话。

澜冰屿试图靠近小夏，但循环把他识别成“未登记人员”，站台边界开始塌陷。旧广播和闸机同时转向他，雾里伸出无数等待登记的人影，要求他也“留在原地等待确认”。

澜冰屿的蓝色环形瞳孔短暂亮起。他用自己作为锚点稳定站台，让小夏不再被循环推回起点。这个动作让他的外套内侧一条坐标布条开始褪色。

澪星无法替人做选择，也不能直接修复异常。它只能把尾灯放低，照亮小夏和失物招领室之间那条很短的路，并用愿望舱临时收住从儿童鞋上剥落的记忆星屑。

澜冰屿稳住空间时，手指边缘短暂变得透明。澪星立刻停下收纳动作，贴近他的靴边，用额头星核轻轻照住他的影子，像是在确认影子还连着他。

## D段：回收遗憾

澜冰屿没有告诉小夏“妈妈很快就来”，也没有承诺能让一切恢复。他只是蹲在她面前，把那只儿童鞋推回到她够得到的位置。

他从碎片里听见母亲真正没能传到的后半句：

```text
别动，站在那里等我。我一定会绕回来。
```

澜冰屿把这句话还给小夏，但不是作为奇迹，而是作为被切断的事实。他告诉她：她不是被忘记了，是那天车站、广播和世界一起断开了。

小夏终于不再盯着冰冷的登记屏。她低头系好鞋带，第一次离开黄线。澪星尾灯照出一条从站台通向失物招领室的路。门后的空间变回临时安置点，墙上出现母亲后来留下的登记信息和一段断裂留言。

小夏听完留言后，身影不再被困在“等待”那一秒。儿童鞋化成一枚小小的蓝灰色记忆碎片，落进澪星的愿望舱包。车站广播终于播放完整：

```text
请看管好您的随身物品和同行儿童。若与家人走散，请前往临时安置点登记。
```

雾散开一点，站台电子屏上的“监护人确认失败”慢慢熄灭，旧指示牌稳定地亮出“临时安置点”。

回收完成后，澜冰屿站在原地，忽然摸向外套内侧。一条手写坐标布条已经完全变白。他知道那里原本写着某个人最后出现的位置，却想不起具体地址，也想不起自己为什么一直带着它。

澪星靠近他，额头星核轻轻碰了碰他的手背。它没有解释，也解释不了。它只是安静地等他。等澜冰屿把手放下来后，澪星又悄悄用尾灯照了一下他的脚边，确认他的影子没有继续变淡。

澜冰屿看着空白布条，把它重新缝回外套内侧，像把一个已经归还的名字送回原处。

## 结尾钩子

清晨前，澜冰屿和澪星离开废弃车站。站外雾气稍薄，远处城市轮廓像沉在水里。

澪星的愿望舱包里，那枚儿童鞋碎片已经安静下来。但在澜冰屿转身时，失物招领窗口后的旧屏幕短暂亮起，显示一行不属于车站系统的文字：

```text
B-07 锚点状态：存在痕迹下降。
```

澜冰屿没有看见这行字。澪星看见了。它的尾灯亮度短暂升高，又很快压低。

最后一个镜头：澜冰屿走入雾中，身影比开场时更淡了一点。澪星停在原地回头，确认他还在，才追上去。
