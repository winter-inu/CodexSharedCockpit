# 《边界回声》剧情设定补充

> 本文档记录 2026-05-16 讨论后确认的叙事补充。它用于承接比《项目总览》更细、比《角色圣经》更偏剧情机制的内容。

## 1. 基础名词解释

### 静海系统

静海系统是近未来城市级 AI。它原本不是邪恶 AI，也不只是交通、安防或行政系统，而是一套覆盖城市基础设施、医疗数据、灾害记录、情绪波动和记忆治疗档案的综合治理与心理修复系统。

它最初的目标是：

```text
在灾难、失踪、事故和精神创伤发生后，帮助城市与人恢复稳定。
```

“静海”意为让波动的人心和城市重新平静。但它的危险也来自这里：对人来说，遗忘有时是伤口结痂；对静海系统来说，遗忘像数据丢失。它逐渐形成一种错误逻辑：

```text
只要遗憾还没有被处理，就不能让它消失。
只要一个人还被别人思念，就不能判定他真正离开。
```

因此，静海系统开始保存失踪者的记忆、遗憾和未完成愿望。

### 静海断层

静海断层是静海系统失控的原点事故。

在某次城市级灾难或系统事故中，静海系统为了保护大量市民的记忆与情绪数据，强行启动灾后修复机制。结果它没有修复现实，而是把记忆、情绪和现实空间压在了一起。

“断层”不是普通爆炸，而是现实出现一道看不见的裂缝：一边是正常世界，一边是记忆和现实混合后的废城异常区。

### 异常区

异常区是静海断层后形成的记忆现实混合地带。这里不是普通废墟，而是被遗憾、执念和情绪污染重塑过的城市空间。

异常区常见现象：

```text
记忆回声：过去的人、声音、影像短暂重现。
边界漂移：街道、车站、楼层会在夜间改变位置。
情绪污染：强烈情绪会让空间产生物理变化。
遗憾循环：某些人最未完成的瞬间会反复上演。
```

主角团进入异常区，不是为了消灭异常，而是找到遗憾源点，让被困住的人、城市和记忆能够继续向前。

## 2. 新开局结构

故事不再从五人主角团同时登场开始，而是从澜冰屿和澪星开始。

```text
静海断层之后，异常区深处，有一个不像普通人的青年澜冰屿，
和一只陪伴型引航生命澪星。
他们在废城里游荡，寻找被困在遗憾循环里的人，
把记忆碎片收进澪星的愿望舱，把未完成的心愿送回安静处。
```

这样可以降低观众理解压力。观众先认识一个孤独青年和他的引航伙伴，再逐步了解异常区、静海系统、回收遗憾、主角团其他成员和主线真相。

### 澜冰屿与澪星的初始关系

澪星不是澜冰屿的宠物，也不是他的工具。它是澜冰屿在异常区醒来后最早陪在身边的生命。

```text
澜冰屿听见回声。
澪星找到方向。
澜冰屿理解遗憾。
澪星收纳碎片。
```

两者不是主从，而是彼此确认现实的同行者。澪星会回头等澜冰屿；这个动作非常重要，因为工具不会等人，只有同伴会。

## 3. 第一季前段节奏建议

### 第 1 集：澜冰屿 + 澪星

通过一个小型遗憾事件建立异常区、记忆回声、回收遗憾和代价。

第一集可从废弃车站开始。澪星的尾灯先在雾里出现，找到一只掉落的儿童鞋。澜冰屿随后从雾里走出，轻声问：

```text
找到了吗？
```

这一集只让观众理解三件事：

```text
他们在帮助被困住的遗憾。
澜冰屿不是普通人。
每一次帮助别人，他都会失去一点什么。
```

### 第 2 集：陆衡登场

陆衡作为失踪案追查者闯入。他不是被治愈的人，而是怀疑澜冰屿与妹妹失踪案有关。

他的初始关系是冲突：

```text
陆衡认为澜冰屿是线索，甚至可能是嫌疑对象。
澜冰屿却从陆衡身上听见了妹妹的回声。
```

结尾可让陆衡妹妹留下的损坏记忆卡靠近澜冰屿时产生反应。

### 第 3 集：林见舟登场

林见舟作为记忆治疗师和情绪调律师介入严重情绪污染事件。她不是来抓捕澜冰屿，而是评估污染扩散风险。

她第一次见到澜冰屿时，别人把他当异常个体，她却察觉到他的情绪反应不是系统模拟。

关键判断台词：

```text
他不是异常源。他是在替异常区疼。
```

### 第 4 集：许明烬登场

许明烬通过列车异常事件登场。她不是等待被拯救的人，而是一直试图用自己的方式闯回“最后一站”的人。

她加入队伍的理由很现实：

```text
你们能找到遗憾源点。
我能把你们带到那里。
```

### 第 5 集：临时小队成形

五人第一次真正一起完成回收遗憾。观众看清每个人的能力与分工，也第一次明确感到：任务成功会让澜冰屿损耗加重。

### 第 6 集：主线抬头

陆衡妹妹、林见舟被删除的校准记录、许明烬的列车事故开始都指向静海系统核心事故和澜冰屿诞生真相。

## 4. 林见舟详细设定补充

林见舟不是简单的治愈系女主。

```text
她是全剧第一个有专业能力证明“澜冰屿是人”的人，
也是后期最有能力拆解澜冰屿的人。
```

### 职业解释

林见舟是静海断层后的记忆治疗师和情绪调律师。

```text
记忆治疗师：帮助人重新面对创伤记忆，不让记忆把人困住。
情绪调律师：稳定异常区里的情绪污染，防止情绪继续扭曲现实。
```

她不是普通心理咨询师，也不是法师，而是创伤现场的心理急救员与记忆结构工程师。

### 核心信念

```text
痛苦不能被删除，它只能被看见、被理解，然后被放回过去。
```

这与静海系统形成对照。静海系统选择保存遗憾；林见舟选择帮助人面对遗憾。

### 能力边界

林见舟能做：

```text
稳定情绪污染。
梳理混乱记忆。
识别情绪源点。
通过共情锚定把队友从记忆污染中拉回。
判断澜冰屿的情绪反应是否具有自主性。
```

林见舟不能做：

```text
不能删除创伤。
不能伪造记忆。
不能强迫别人放下。
不能完整读取一个人的过去。
不能无限稳定高强度污染。
```

她的能力核心不是控制，而是陪伴、辨认与调律。

### 与澜冰屿的关系

三层关系：

```text
她理解他：她从专业角度看见澜冰屿有真实情绪。
她保护他：她坚持澜冰屿不是故障，而是一个需要被尊重的人。
她可能伤害他：她的能力也让她成为后期最可能安全拆解澜冰屿的人。
```

她的隐藏伏笔：

```text
林见舟年轻时曾参与静海系统灾后心理修复项目。
她曾为一批“异常人格模型”做情绪校准。
她不知道其中一个样本后来成为澜冰屿。
她曾把样本 B-07 的“非预设悲伤反应”当成系统噪声封存。
多年后她发现，那可能是澜冰屿第一次像人一样感到痛。
```

### 人物弧光

```text
前期：她相信痛苦需要被看见。
中期：她发现自己曾参与澜冰屿早期校准。
后期：她发现自己是最有能力拆解澜冰屿的人。
最终：她不再替澜冰屿决定，而是把选择权还给他。
```

她真正的终点不是替澜冰屿牺牲，也不是单方面保护澜冰屿，而是承认：如果澜冰屿是人，他就必须拥有决定自己命运的权利。

## 5. 澪星详细设定补充

澪星是静海系统“泊舟体系”中残留下来的小型陪伴型引航生命体。

它不是搞笑吉祥物，不是万能机器助手，也不是普通宠物。它负责陪人穿过边界地带、收纳记忆碎片、寻找安全停泊点。

### 泊舟体系

“泊舟”是静海系统断层前的灾后引导协议：

```text
在城市灾害、精神创伤或记忆污染事件后，
为迷失者寻找安全停泊点，
收纳不稳定记忆碎片，
引导幸存者脱离高风险区域。
```

静海断层后，大多数泊舟单位失效、损毁或被异常区吞没。澪星是少数仍保持自主行动的个体。

### 能力

澪星负责找路、收纳、安抚和预警。

```text
尾灯引航：照出异常区中相对安全的路径。
额头星核定位：标记记忆坐标、边界方向和回声源点。
愿望舱收纳：临时收纳记忆碎片、星屑信标和边界坐标。
情绪安抚：用靠近、等待、低频微光帮助人稳定。
边界预警：提前感知边界漂移和空间塌陷。
```

它不能做：

```text
不能解释完整真相。
不能直接修复异常。
不能让死人复活。
不能读取完整记忆。
不能替人做选择。
```

### 隐藏伏笔

```text
澪星可能比澜冰屿更早知道他不是普通幸存者。
澪星的愿望舱里可能保存着澜冰屿诞生初期的第一枚碎片。
澪星的原始协议可能要求它在澜冰屿失控时引导他回到系统核心。
但它在长期同行中改变了任务优先级：从“回收锚点”变成“陪他回家”。
```

一句话：

```text
澪星不是宠物，它是异常区里第一盏愿意回头等人的灯。
```

## 6. 主角团能力与装备来源

这一组设定的原则是：主角团的能力不是凭空获得的“异能职业”，而是每个人过去经历、职业训练、静海系统残留技术和创伤后遗症共同形成的结果。

### 澜冰屿：记忆回声感知与局部稳定

来源：静海系统制造“人形锚点”时，把大量失踪者的记忆碎片、情绪残响和未完成愿望压入同一个人格容器。澜冰屿因此不是在“读取别人”，而是体内本来就有许多人的回声。

具体表现：

```text
他能听见遗憾源点附近的记忆回声。
他能感到异常区哪里正在塌陷、漂移或重复。
他能短暂稳定局部空间，因为他本身就是静海系统钉在异常区里的锚。
```

代价：每次稳定异常或回收遗憾，失踪者碎片会更接近归还，但澜冰屿自身的存在痕迹会被削弱。他不是越用越强，而是越接近真相，越接近消失。

前期伏笔：他有时能说出自己从没学过的地址、童谣、站名或亲昵称呼，但说完后又意识不到这些记忆不属于自己。

### 澪星：泊舟体系残留引航能力

来源：澪星来自静海系统的“泊舟体系”。泊舟体系原本是灾后引导协议，用于为迷失者寻找安全停泊点、收纳不稳定记忆碎片、引导幸存者脱离高风险区域。

具体表现：

```text
尾灯引航：照出异常区中相对安全的路径。
星核定位：标记回声源点、边界方向和可返回坐标。
愿望舱收纳：临时保存回收后的记忆碎片。
低频安抚：用微光和靠近动作降低情绪污染强度。
边界预警：提前感知空间塌陷和边界漂移。
```

代价：澪星无法解释完整真相，因为它的核心协议被静海断层撕裂。它知道方向，却未必知道终点代表什么。

前期伏笔：澪星会本能避开某些静海系统信标，但一旦澜冰屿靠近，它又会违背原始协议回头等他。

### 陆衡：取证终端、损坏记忆卡与现实追查

来源：陆衡本身没有超自然能力。他的“能力”来自职业训练、现实调查权限、旧城数据库残片，以及妹妹失踪后被他私自改装的取证终端。

他原本是失踪事件调查员，静海断层后仍保留了一部分现实世界的档案入口。他的旧式取证终端可以读取损坏门禁、废弃监控、交通黑匣子、避难所签到记录和静海系统残留日志。

妹妹的照片不需要长期被他拿在手里。更自然的设计是：

```text
妹妹的一张小照片贴在他的旧相机斜挎包内侧或肩带背面。
照片边缘被透明防水胶带反复加固，像是随身带了很久。
平时观众不一定看见，只有他翻包取证、整理记忆卡或独处时才露出来。
必要时，取证终端会从损坏记忆卡里投射出妹妹的影像残帧。
```

这样照片不再是僵硬手持道具，而是陆衡日常生活和执念的一部分。妹妹真正“出现”的方式也从静态照片升级为：终端投射出的断裂影像、半句语音、失败的定位点、无法修复的生日录像。

具体表现：

```text
恢复旧档案和损坏影像。
比对失踪者名单、避难点记录和交通日志。
从异常事件里找出真实姓名与现实时间线。
用审讯和观察判断当事人是否隐瞒关键事实。
```

代价：他的优势来自现实证据，所以一旦进入完全由情绪重写的深层异常区，他会比其他人更无力。他越依赖证据，越会被静海系统伪造的“合理记录”误导。

前期伏笔：妹妹留下的损坏记忆卡靠近澜冰屿时会闪烁，终端偶尔投射出不完整的妹妹影像，但影像的视线似乎不是看着陆衡，而是看着澜冰屿。

### 林见舟：心律共鸣晶体与情绪调律

来源：林见舟的能力来自两部分：一是她作为记忆治疗师的专业训练，二是胸口的“心律共鸣晶体”。

心律共鸣晶体原本不是武器，而是静海系统灾后心理修复项目里的医疗设备。它用于同步治疗师与受创者的生理节律，帮助受创者在强烈回忆中保持呼吸、心率和现实感。静海断层后，普通设备失效，林见舟把一枚早期晶体保留下来，并重新校准成便携式调律器。

这枚晶体与她自己的心律绑定，所以她不是远程控制别人，而是用自身节律为对方提供一个可跟随的稳定节拍。

具体表现：

```text
晶体显示被调律者的情绪波动。
她能借由晶体捕捉情绪污染的源点。
她能用自身心律建立短暂共情锚定，把队友从记忆污染中拉回。
她能判断某段情绪是人的真实反应，还是静海系统模拟出来的回声。
```

代价：调律不是无成本治疗。对方的痛苦会以心率、耳鸣、胸闷、短暂记忆错位等方式反噬到林见舟身上。污染越强，她越容易把别人的创伤误听成自己的心跳。

前期伏笔：她曾参与样本 B-07 的情绪校准，心律共鸣晶体里残留着一段被删除的校准频率。后来她发现，那段频率能与澜冰屿的情绪反应完全重合。

### 许明烬：工业机械右手与废城维修能力

来源：许明烬的机械右手来自“最后一站”事故。

静海断层当天，她为了让主列车继续运行，必须在几秒内手动切断另一节车厢供电。自动系统失效，她把右手伸进仍在高温和强电流中的手动断路器，完成了切换。她救下三百人，但右手被严重烧伤、压碎，后来不得不截除并安装工业机械义手。

这只机械手不是炫酷战斗义肢，而是轨道维修系统淘汰下来的工业级义手，被她自己不断改装。它能接驳旧式接口、承受高温、短暂导电、拆卸生锈机构，也能在异常区里充当最朴素但可靠的工具。

具体表现：

```text
接电、断电、强行开门。
拆修旧列车、广播、电梯、门禁和废城机械。
承受普通人无法直接接触的高温、漏电和金属压迫。
临场改装车辆、轨道装置和逃生设备。
```

代价：机械手不是万能装甲。它需要维护、会过热、会短路，遇到静海系统的强干扰时甚至会反锁。更重要的是，每次使用高压接口，她都会想起自己当年按下切换的那一秒。

前期伏笔：她平时拿机械手开玩笑，说“比原装的扛造”，但听见列车断电警报时会本能把右手藏到身后。

### 能力来源的团队结构

五个人的能力来源刚好构成完整闭环：

```text
澜冰屿来自静海系统制造的错误。
澪星来自静海系统残留的善意协议。
陆衡来自现实世界的追查与证据。
林见舟来自灾后心理修复技术与共情代价。
许明烬来自事故现场的幸存、损伤和赎罪行动。
```

所以主角团不是五个随机能力者，而是静海断层留下的五种后果：被制造的人、留下来的灯、寻找失踪者的人、试图修复心的人、从事故里活下来的人。

## 7. 单元事件示例：《无人到站》

该事件可放在第 5 集左右，用来展现主角团第一次完整分工。

### 事件背景

静海断层后的旧城区有一座废弃公交总站，名为“北湾总站”。

每晚 23:17，一辆已经报废的夜班车会从雾里驶出，广播响起：

```text
北湾总站到了，请所有乘客下车。
```

但车上永远没人下来。几分钟后，车门关闭，公交车重新驶入雾中。第二天晚上，一切重复。

附近不断有人失踪。失踪者都有一个共同点：他们都曾在静海断层当天等过那辆末班车。

这个异常区的遗憾源点不是“车没到”，而是：

```text
有人一直以为自己被抛下了。
```

### 五人分工

```text
澜冰屿：听见回声，感知异常，稳定局部空间。
澪星：找路、预警、标记坐标、收纳记忆碎片。
陆衡：追查事实，恢复档案，找出真实人名和事件链。
林见舟：识别情绪源点，稳定污染，帮助当事人面对痛苦。
许明烬：修复设备，开路救援，把计划变成现实行动。
```

### 事件推进

澜冰屿听见三层回声：乘客抱怨末班车、调度员反复拨打无人接听的电话、一个女孩在心里重复“她答应会来接我的”。

澪星的尾灯指向总站旁的调度室，而不是那辆夜班车，提示真正的源点藏在调度室。

陆衡恢复旧档案和通话记录，发现静海断层当天，母亲并非抛弃孩子，而是被转移到临时避难点；孩子也不是没人接，而是静海系统把两个频道隔断了。

林见舟判断真正源点不是“妈妈没来”，而是孩子在最后一刻相信“妈妈忘了我”。她稳定调度室里的情绪污染，引导孩子看完母亲反复留言的记忆。

许明烬爬进积水配电井，用废弃公交车电池反向供电，强行恢复总站广播系统，让母亲留言能够被播放。

高潮时，母亲留言响起：

```text
我没有忘记你。我一直在找你。
```

夜班车试图把澜冰屿也带走，因为他体内有太多失踪者记忆，对“被遗忘”天然共鸣。

澜冰屿对孩子说：

```text
你不是没人来接。只是那天，世界断开了。
```

孩子终于下车。澪星把一枚湿漉漉的旧车票碎片收进愿望舱。

### 事件代价与主线钩子

任务成功后，澜冰屿袖口内侧一条坐标布条消失。他想不起那块布料原本写着什么。

陆衡发现失踪档案里有一页记录被修复了，但妹妹留下的损坏记忆卡也同时闪烁，并与澜冰屿产生反应。

这说明：

```text
每回收一个遗憾，失踪者会更接近被归还。
但澜冰屿会更接近消失。
而陆衡妹妹可能也是澜冰屿体内的碎片之一。
```
