# 中文文件命名规则

## 适用范围

本项目《边界回声》后续新建文件默认使用中文命名。

适用对象：

- 剧本
- 大纲
- 分镜表
- 视频提示词
- 关键帧提示词
- 审核记录
- 修正方案
- 角色海报
- 关键帧图片
- 资产登记

## 命名原则

- 文件名使用中文描述内容。
- 可以保留必要制作编号，如 OP01、V1、V2、v3。
- 文件名尽量短，但必须能看出用途。
- 同一批文件使用统一格式。
- 不再新建全英文制作文件名。

## 推荐格式

### 文档

```text
第一集_OP01-OP09_宫格分镜表_V1.md
第一集_OP01-OP09_图生视频_无口型_台词时间槽_V2.md
第一集_OP07-OP12_分镜与关键帧_V1.md
第一集_V4_完整剧本_最新版.md
```

### 关键帧图片

```text
镜头01_午后车站大全景_v2.png
镜头02_正面中景_妈妈牵小夏行走_v4.png
镜头09_右脚鞋掉落_v2.png
```

### 角色资产

```text
澜冰屿_角色海报.png
林见舟_角色海报.png
主角团_群像海报.png
```

## 特例

技术入口文件可保留行业默认名称，例如：

```text
index.html
```

原因是部分本地网页或服务器会默认识别该文件名。除这类技术入口外，制作资料与资产文件均使用中文命名。
