# 中文文件命名规则

## 适用范围

本项目《边界回声》后续新建制作文件默认使用中文命名。

适用对象：

- 剧本
- 大纲
- 分镜表
- 视频提示词
- 关键帧提示词
- 审核记录
- 修正方案
- 角色海报
- 关键帧图片
- 资产登记

## 命名原则

- 文件名使用中文描述内容。
- 可以保留必要制作编号，如 EP01、OP01、V5、V6。
- 文件名尽量短，但必须能看出用途。
- 同一批文件使用统一格式。
- 第一集当前只保留 V5 制作基准，不再新建或引用旧版命名文件。

## 推荐格式

### 文档

```text
第一集_V5_完整剧本_最新版.md
第一集_V5_情绪版大纲.md
第一集_V5_宫格分镜表.md
第一集_V5_图生视频制作清单.md
```

### 关键帧图片

```text
V5_OP08-2_P0长椅正字刻痕特写_关键帧_v1.png
V5_OP11-4_妈妈保护性松手_关键帧_v1.png
V5_OP22-2_澜冰屿影子变淡_关键帧_v1.png
```

### 角色资产

```text
澜冰屿_角色海报_v2.png
澪星_角色海报_v2.png
小夏鞋子_第一集统一参考_v1.png
```

## 技术入口例外

技术入口文件可保留行业默认名称，例如：

```text
index.html
README.md
```

原因是本地网页或服务器会默认识别这些文件名。除这类技术入口外，制作资料与资产文件均优先使用中文命名。
