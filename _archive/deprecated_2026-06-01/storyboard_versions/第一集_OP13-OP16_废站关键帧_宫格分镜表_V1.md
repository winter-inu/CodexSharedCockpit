# 第一集 OP13-OP16 废站关键帧宫格分镜表 V1

当前已完成 OP13 宫格关键帧，但 OP12-6 到 OP13-1 的背景连续性需要重做锁定。OP13 承接 OP12-6 的鞋子特写：镜头位置、视角、构图、鞋在画面中的位置、地面透视线、背景大结构都不能变化，只能让同一画面经历多年时间腐蚀。事故当天刚掉落的鞋转入多年后雨夜废弃旧北站，再由澪星发现鞋、澜冰屿画外音入场，最后给到澜冰屿从黑暗中正面出场、主观视角走到鞋前，并切入 OP14 的侧面半蹲观察。

废弃旧北站发生静海断层事故多年后已经年久失修、破旧、冷清、荒废、完全断电。OP13-OP16 废站段不要任何仍在工作的车站灯、顶灯、屏幕光、闸机灯、红色状态灯、广告牌光或霓虹；可见光只允许来自雨夜极弱天光 / 冷月光、湿地反光、澪星引航灯和极微弱等待回声光。

## OP12-6 到 OP13-1 同镜头腐蚀锁

- OP13-1 不是新镜头，不是换到另一个废站角度，而是 OP12-6 原图的多年后状态。
- 必须保持 OP12-6 的固定低机位鞋子特写：同一镜头高度、同一镜头焦段感、同一透视方向、同一鞋子位置、同一地面砖缝 / 水泥缝方向、同一背景大轮廓。
- 只允许变化：鞋从事故当天干净状态变成潮湿、破旧、褪色、沾灰；地面从事故当天站厅地面变成同一位置多年后积水、裂缝、灰尘、锈迹、杂物；背景结构从事故当天车站变成同一结构的断电废弃状态。
- 不允许变化：不换视角、不换机位、不重新设计背景、不新增另一组闸机 / 长椅 / 站台结构、不把鞋挪到画面别处、不把地面透视方向改掉。
- 生成 OP13-1 时应优先使用 OP12-6 作为参考图做图生图 / 续图，而不是纯文字重新生成。

澪星造型参考已切换为 V2 版海报：

- 主参考：`assets/characters/澪星_角色海报_v2.png`
- 设定板：`assets/characters/澪星_角色海报_v2_设定板.png`
- 与澜冰屿同框时，澪星高度约到澜冰屿膝盖，不是脚踝高度的小玩具。
- 尾灯方向：正常状态下，引航灯底部与尾巴相连，顶部小星星朝上；尾巴伸到身前照亮鞋子时，引航灯倒置，底部在上方连接尾巴，顶部小星星朝下。

澜冰屿造型参考已切换为 V2 版海报，但剧情关键帧必须使用项目统一写实 CG 风格，不使用动画或 anime 正向词：

- 主参考：`assets/characters/澜冰屿_角色海报_v2.png`
- 设定板：`assets/characters/澜冰屿_角色海报_v2_设定板.png`
- 统一风格：高精度写实 CG 渲染风 / 电影级关键帧插画风，人物风格对齐 OP12 小夏和妈妈，不做动画男主、漫画美少年或游戏角色立绘。
- 发色比例：澜冰屿参考 V2 版海报2，短黑发为主，黑发层里零碎散落少量白色发丝；白色部分细碎、断续、随机分布，不是一束一束的白色挑染，也不是半头白发或大片银白刘海。
