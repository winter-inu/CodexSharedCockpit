```text
请阅读 IMPORT_INBOX.md，把其中与“边界回声”相关的内容整理为：
1. 项目目标
2. 世界观/设定
3. 角色与关系
4. 已确认决策
5. 未解决问题
6. 当前任务拆分
7. 下一步给 Codex 的执行任务

然后更新 项目上下文.md、CONVERSATION_LOG.md、决策记录.md、任务看板.md 和 同步包.md。
```

## 澪星尾灯 Prompt 规则

```text
澪星尾灯不是可拆卸道具，不能被其他角色拿走。

待机状态：
尾灯贴近尾部，无明显蓝色细线。

指路 / 照明状态：
尾灯轻微悬浮展开，可见短蓝色能量牵引线。

校准 / 回声定位 / 收纳状态：
尾灯光线更明显，蓝色细线可延伸，但不能像武器、激光炮或魔法鞭。
```
