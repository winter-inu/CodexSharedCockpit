# 角色视觉参考.md

## 用途

这里记录《边界回声》主角团当前共享视觉参考图。ChatGPT、Codex 和图像生成工具在讨论或生成角色相关画面时，应优先参考这些文件，避免仅凭文字设定导致外观漂移。

## 使用规则

- 角色外观以 `边界回声_角色圣经.md` 和本文件中的参考图共同为准。
- 生成关键帧时，不要重新设计角色外观。
- 澪星尤其必须参考 `assets/characters/澪星_角色海报.png`，不要改成兔耳、普通玩偶、蓝白猫型机器人、红项圈、铃铛或腹部圆口袋。
- 陆衡使用 `陆衡_角色海报_v2.png` 作为当前新版视觉参考。
- 许明烬使用 `许明烬_角色海报_v2.png` 作为当前新版视觉参考。
- 如果未来替换角色定稿图，必须更新本文件和 `变更记录.md`。

## 给 ChatGPT 的提示词

```text
请先读取 角色视觉参考.md，并把 assets/characters 下的角色海报作为当前角色外观参考。后续生成关键帧、海报或分镜时，不要只按文字设定猜角色外观。
```
