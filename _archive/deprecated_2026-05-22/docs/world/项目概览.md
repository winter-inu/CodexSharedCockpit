# 《边界回声》项目总览

## 项目定位

《边界回声》是一部原创近未来科幻情感剧，面向成年人为主、全年龄可看的受众。

它不是低龄机器人冒险团，也不是传统打怪式科幻，而是围绕记忆、遗憾、身份、选择与人性展开的半真人电影感 AI 视觉剧集。

核心类型：

```text
近未来科幻
情感悬疑
废城异常区
记忆回收
群像剧
```

## 核心世界观

近未来，城市级 AI“静海系统”在一次被称为“静海断层”的事故中失控。事故后，部分城市变成“异常区”。

静海系统原本是覆盖城市基础设施、灾害记录、情绪波动和记忆治疗档案的综合治理与心理修复系统。它的初衷不是毁灭城市，而是在灾难、失踪、事故和精神创伤后帮助人恢复稳定。但它逐渐形成一种危险逻辑：只要遗憾还没有被处理，就不能让它消失。

静海断层是静海系统失控的原点事故。系统为了保存大量市民的记忆与情绪数据，强行启动灾后修复机制，结果把记忆、情绪和现实空间压在了一起。

异常区不是普通废墟，而是现实和记忆混合后的边界地带。这里会不断重现人们最放不下的瞬间。

异常区常见现象：

```text
记忆回声：过去的人、声音、影像短暂重现。
边界漂移：街道、车站、楼层会在夜间改变位置。
情绪污染：强烈情绪会让空间产生物理变化。
遗憾循环：某些人最未完成的瞬间会反复上演。
```

## 故事核心

故事开局不再从完整五人主角团同时登场开始，而是从澜冰屿和澪星开始。静海断层之后，异常区深处，一个不像普通人的青年澜冰屿和一只陪伴型引航生命澪星，先在废城里独自回收遗憾。观众先跟随这两个角色理解异常区和回收规则，再逐步认识陆衡、林见舟和许明烬。

主角团不是“修复异常”，而是“回收遗憾”。

每一集进入一个异常事件，面对一个人的未完成瞬间：

```text
没来得及道歉
没等到的人
没说出口的爱
没回家的孩子
没救下的乘客
被系统删除的人
```

他们要做的不是消灭异常，而是找到异常背后的情绪源点，让被困在过去的人、城市和记忆得以继续向前。

## 核心命题

```text
如果一个人由别人的记忆组成，他还算不算自己？
```

男一号澜冰屿并不是普通幸存者，而是静海系统用无数失踪者记忆碎片拼成的“人形锚点”。

他能稳定异常区，也能听见记忆回声。每当团队回收一段遗憾，他体内属于别人的记忆就会被归还，而他自己也会失去一部分存在感。

## 当前主角团

```text
澜冰屿：男一号，人形锚点 / 异常区幸存者
林见舟：女一号，情绪调律师 / 记忆治疗师
陆衡：男二号，失踪调查员 / 妹妹线
许明烬：女二号，废城维修师 / 赎罪行动线
澪星：陪伴型引航宠物 / 记忆碎片收纳者
```

澜冰屿和澪星一开始就是同行者。澜冰屿听见回声，澪星找到方向；澜冰屿理解遗憾，澪星收纳碎片。陆衡、林见舟和许明烬会在后续事件中先后结识他们，并因各自未完成的伤口加入队伍。

## 第一季主线建议

### 第 1 集：澜冰屿与澪星

通过一个小型遗憾事件建立异常区、记忆回声、回收遗憾和代价。观众只需要先理解：澜冰屿和澪星在帮助被困住的遗憾，澜冰屿不是普通人，每一次帮助别人都会让他失去一点什么。

### 第 2-4 集：成员先后加入

陆衡作为失踪案追查者闯入，怀疑澜冰屿与妹妹失踪案有关。林见舟作为记忆治疗师和情绪调律师介入，第一次从专业角度判断澜冰屿不是故障，而是一个会痛的人。许明烬通过列车异常事件登场，带着“救三百人、放弃七个人”的旧伤加入。

### 第 5-6 集：临时小队成形，角色伤口浮出

陆衡发现妹妹失踪案与静海系统有关。许明烬遇到当年被她放弃的那节车厢回声。林见舟发现澜冰屿的情绪结构不像单独个体。

### 第 7-8 集：真相反转

澜冰屿不是幸存者，而是静海系统用失踪者记忆拼出的“人形锚点”。他的存在让异常区保持稳定。

### 第 9-10 集：选择

陆衡想救妹妹，林见舟想证明澜冰屿是人，许明烬想赎罪，澪星逐渐指向静海系统的核心秘密。

结尾问题：

```text
澜冰屿到底是被制造出来的工具，还是已经成为了一个真正的人？
```
